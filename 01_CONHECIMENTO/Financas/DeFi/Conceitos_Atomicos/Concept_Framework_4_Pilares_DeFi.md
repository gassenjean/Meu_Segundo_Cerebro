# Framework: 4 Pilares da Análise Fundamentalista

**Fonte:** Módulo 4 (Análise Científica)
**Autor:** Lucas Amoedo
**Contexto:** Avaliação de Projetos DeFi

## ESTRUTURA

### 1. Utilidade + Distribuição Justa

* O token resolve um problema real?
* A distribuição pública é >40%? (Evitar concentração em VCs).
* Existe controle de inflação?

### 2. Demanda Real > Subsídios

* A receita do protocolo vem de usuários pagantes ou de emissão de tokens (inflação)?
* Use **Token Terminal** para verificar receita real.

### 3. Narrativa + Espaço de Crescimento

* O mercado endereçável (SAM/TAM) é grande?
* O projeto está alinhado com a narrativa macro do ciclo?
* Existe desenvolvimento ativo (GitHub)?

### 4. Verificação Independente

* **TinEye:** Fotos dos fundadores são reais?
* **Etherscan:** Contratos verificados?
* **Exchanges:** Liquidez em múltiplos locais?
