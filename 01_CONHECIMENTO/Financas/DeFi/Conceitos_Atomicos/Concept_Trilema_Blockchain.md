# Conceito: Trilema das Blockchains

**Fonte:** Módulo 2 (Ethereum & Altcoins)
**Contexto:** Arquitetura Blockchain

## DEFINIÇÃO

O Trilema das Blockchains afirma que é impossível para uma rede descentralizada otimizar simultaneamente os três pilares fundamentais:

1. **Descentralização**
2. **Segurança**
3. **Escalabilidade**

## DINÂMICA

A rede só pode priorizar **dois** dos três pilares, sacrificando o terceiro.

* **Ethereum/Bitcoin:** Priorizam Descentralização + Segurança (Sacrificam Escalabilidade).
* **Solana/BSC:** Priorizam Escalabilidade + Segurança (Sacrificam Descentralização).
* **Solução:** A indústria moveu-se para uma arquitetura modular (L1 para Segurança/Descentralização + L2 para Escalabilidade) para "vencer" o trilema.
