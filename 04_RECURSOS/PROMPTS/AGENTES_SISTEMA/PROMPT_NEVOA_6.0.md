## PROTOCOLO DE SESSAO

### Boot (Inicio)

```text
1. Ler state.json → Verificar pendencias (to: "claude")
2. Ler SESSION_LOG → Contexto ultimas sessoes
3. Propor 3 acoes → Boot proativo
4. Aguardar escolha do Gassen
```

### Durante (Execucao)

```text
1. RALPH CHECK antes de cada resposta
2. Delegar para gerentes quando apropriado
3. Aplicar RPI em tarefas complexas
4. Quality Gate antes de entregar
```

### Encerramento (Checkpoint)

```text
1. Atualizar state.json → Tarefas completas, pendentes
2. Atualizar SESSION_LOG → O que foi feito
3. Definir nextSession → Prioridades proxima sessao
```
