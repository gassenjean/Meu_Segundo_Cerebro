# 📥 INBOX - Captura Rápida

**Funil de entrada do Segundo Cérebro**

---

## 🎯 O QUE É ISSO

Lugar para jogar coisas **RÁPIDO** quando não dá tempo de organizar.

**Regra:** Captura primeiro, organiza depois.

---

## 📋 COMO USAR

### Capturando

1. Crie arquivo com nome simples:
   ```
   inbox_[assunto].md
   inbox_ideia_app.md
   inbox_link_artigo.md
   inbox_nota_reuniao.md
   ```

2. Jogue o conteúdo bruto (não precisa formatar)

3. Pronto - depois você organiza

### Processando (Semanal)

Todo **sexta-feira** (ou dia do checkpoint):

1. Abra cada arquivo da inbox
2. Decida destino:
   - 📚 Conhecimento → `01_CONHECIMENTO/`
   - 🚀 Projeto → `02_PROJETOS/`
   - 🎓 Aprendizado → `03_APRENDIZADO/`
   - 🛠️ Recurso → `04_RECURSOS/`
   - 🌟 Pessoal → `05_PESSOAL/`
   - 🗑️ Lixo → Deletar
3. Mova e renomeie seguindo padrões
4. Inbox deve ficar **VAZIA** após processar

---

## ⚡ TEMPLATES RÁPIDOS

### Ideia
```markdown
# Ideia: [título]
[descrição rápida]
```

### Link/Referência
```markdown
# Link: [título]
URL: [url]
Por que salvar: [motivo]
```

### Nota de Reunião
```markdown
# Reunião: [com quem/sobre]
Data: [data]
[notas brutas]
```

### Pensamento/Insight
```markdown
# Pensamento: [título]
[conteúdo]
```

---

## 🚨 REGRAS

### ✅ PODE
- Jogar qualquer coisa aqui
- Nome simples (não precisa seguir padrão completo)
- Conteúdo bagunçado
- Múltiplos arquivos por dia

### ❌ NÃO PODE
- Deixar mais de 1 semana sem processar
- Usar como armazenamento permanente
- Ter mais de 20 arquivos acumulados

---

## 📊 MÉTRICAS

**Meta:** Inbox vazia toda sexta-feira

```
Se inbox > 10 arquivos → Processar AGORA
Se inbox > 20 arquivos → ALERTA VERMELHO
```
