---
description: Ativar Contexto de Aprendizado
---

1. **Carregar Contexto de Aprendizado**
   - Leia o arquivo `03_APRENDIZADO/_MOC_Aprendizado.md` para entender o estado atual dos estudos.
   - Leia `00_SISTEMA/PADROES/NOMENCLATURA.md` se precisar criar novos arquivos.
   - Leia `04_RECURSOS/TEMPLATES/TEMPLATE_Nota_Curso.md` para usar como base para novas notas.

2. **Identificar Objetivo**
   - Pergunte ao usuário: "Qual curso você quer estudar agora?" se não foi especificado.

3. **Executar Sessão de Estudo**
   - Se for criar notas, use o padrão `Curso_[Nome]_[Modulo]_[Aula].md`.
   - Mantenha o foco em criar notas estruturadas com resumo de pontos chave.
   - **NUNCA** misture com contexto de projetos.

4. **Finalizar**
   - Atualize o `_MOC_Aprendizado.md` com o progresso.
