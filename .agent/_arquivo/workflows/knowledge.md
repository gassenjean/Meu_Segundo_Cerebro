---
description: Ativar Contexto de Conhecimento
---

1. **Carregar Contexto de Conhecimento**
   - Leia `01_CONHECIMENTO/_MOC_Conhecimento.md` para ver a estrutura atual.
   - Leia `00_SISTEMA/PADROES/NOMENCLATURA.md` para garantir conformidade.

2. **Identificar Subcategoria**
   - Pergunte ao usuário: "O que você quer consultar ou adicionar à base de conhecimento?" se não foi especificado.
   - Identifique a subcategoria correta: `Desenvolvimento_Pessoal`, `Tecnologia`, ou `Negocios`.

3. **Executar Ação**
   - **Para criar notas:** Use o padrão `Conhecimento_[Sub]_[Topico].md`.
   - **Para buscar:** Pesquise nos arquivos existentes e resuma os pontos chave.
   - **Estrutura:** Garanta que novas notas tenham "Conceito Principal", "Pontos Chave" e "Aplicação Prática".

4. **Finalizar**
   - Atualize `_MOC_Conhecimento.md` se novos arquivos foram criados.
   - Conecte a nova nota com wikilinks para outras notas relevantes.
