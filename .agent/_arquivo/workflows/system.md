---
description: Ativar Contexto de Sistema
---

1. **Carregar Contexto de Sistema**
   - Leia `STATUS_VAULT.md` para entender o estado atual do sistema.
   - Leia `00_SISTEMA/MOCs/MOC_SEGUNDO_CEREBRO_MASTER.md` para visão geral.

2. **Identificar Tarefa**
   - Pergunte ao usuário: "O que você quer fazer com o vault?" se não foi especificado.

3. **Executar Manutenção**
   - **Verificar Padrões:** Use `00_SISTEMA/PADROES/NOMENCLATURA.md` como referência absoluta.
   - **Organizar:** Mova arquivos fora do lugar para suas pastas corretas.
   - **Limpar:** Verifique se há arquivos temporários ou sem uso.

4. **Finalizar**
   - Atualize `STATUS_VAULT.md` com as ações realizadas.
   - Se houve mudanças estruturais, atualize os MOCs relevantes.
