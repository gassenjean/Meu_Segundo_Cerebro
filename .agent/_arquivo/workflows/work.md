---
description: Ativar Contexto de Projetos
---

1. **Carregar Contexto de Projetos**
   - Leia `02_PROJETOS/_MOC_Projetos.md` para ver a lista de projetos ativos.
   - Leia `00_SISTEMA/PADROES/ESTRUTURA_PROJETOS.md` para garantir conformidade com a estrutura.

2. **Identificar Projeto**
   - Pergunte ao usuário: "Qual projeto você quer trabalhar agora?" se não foi especificado.

3. **Verificar Estado do Projeto**
   - Antes de qualquer alteração, leia o `STATUS_ATUAL.md` do projeto escolhido.

4. **Executar Trabalho**
   - Siga rigorosamente a estrutura de pastas: `docs/`, `planejamento/`, `tarefas/`, etc.
   - Não crie arquivos na raiz do projeto, exceto `README.md` e `STATUS_ATUAL.md`.

5. **Finalizar**
   - Atualize o `STATUS_ATUAL.md` do projeto com as mudanças realizadas.
   - Se houve mudança de status macro, atualize `_MOC_Projetos.md`.
