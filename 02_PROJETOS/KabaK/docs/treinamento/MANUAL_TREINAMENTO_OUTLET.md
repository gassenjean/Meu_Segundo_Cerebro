# Manual de Treinamento - Outlet KabaK

**Bem-vindo ao time KabaK!**

Este manual foi criado para ajudar voce a entender a marca, os produtos e como atender nossos clientes da melhor forma. Use como guia de consulta sempre que precisar.

---

## 1. Sobre a KabaK

### Nossa Historia

A KabaK nasceu de uma familia que trabalha com moda ha mais de 20 anos. Temos fabrica propria em Minas Gerais e conhecemos cada etapa da producao - do tecido ate a peca pronta.

Comecamos vendendo para outras lojas e agora estamos expandindo para vender direto ao consumidor. Voce faz parte dessa nova fase!

### Nossos Valores

| Valor | O Que Significa na Pratica |
| ----- | -------------------------- |
| **Qualidade** | So vendemos o que usariamos |
| **Preco Justo** | Sem intermediarios = economia pro cliente |
| **Honestidade** | Se nao temos, dizemos. Se erramos, consertamos |
| **Respeito** | Todo cliente merece atencao, independente do quanto vai comprar |

### Nosso Diferencial

Por que a KabaK e diferente?

1. **Fabrica propria** - Controlamos a qualidade do comeco ao fim
2. **Tecido importado** - Material de primeira linha
3. **Preco de fabrica** - Sem intermediarios, o cliente economiza
4. **Modelagem brasileira** - Feito pensando no corpo brasileiro

## 2. O Outlet KabaK

### O Que E o Outlet

O Outlet e nossa loja fisica na rodovia. Aqui vendemos:

- Pecas masculinas
- Moda praia
- Estoque de colecoes anteriores
- Pecas com pequenos defeitos (informar ao cliente)

**Importante:** O Outlet e separado do e-commerce. Sao marcas diferentes:

| Canal | Foco | Publico |
| ----- | ---- | ------- |
| E-commerce KabaK | Fitness feminino premium | Mulheres classe B/C |
| Outlet | Masculino, praia, queima | Todos que passam na rodovia |

### Por Que Existimos

1. **Queima de estoque** - Pecas que nao venderam online
2. **Produtos masculinos** - Linha diferente do e-commerce
3. **Moda praia** - Aproveitar fluxo de viajantes
4. **Gerar caixa** - Faturamento rapido enquanto o e-commerce cresce

### Nosso Publico

Quem passa pela rodovia:

- Familias viajando (compram presentes)
- Caminhoneiros (procuram conforto)
- Turistas (querem lembrancinhas/moda praia)
- Moradores da regiao (voltam sempre)

**Dica:** Observe de onde o cliente vem. Se esta viajando, pode querer comprar mais de uma vez (presentes).

## 4. Atendimento ao Cliente

### Abordagem Inicial

**Momento certo:** Espere o cliente entrar e olhar por 10-15 segundos antes de abordar.

**O que dizer:**

| Situacao | Abordagem |
| -------- | --------- |
| Cliente entrando | "Boa tarde! Seja bem-vindo ao KabaK Outlet!" |
| Cliente olhando arara | "Esses aqui sao tamanho M, se precisar de outro tamanho me avisa!" |
| Cliente com crianca | "Oi! Temos area infantil ali no fundo, se precisar!" |

**O que NAO fazer:**

- Abordar antes do cliente respirar
- Perguntar "Posso ajudar?" (resposta quase sempre e "so olhando")
- Seguir o cliente pela loja
- Ficar encarando

### Como Apresentar Produtos

**Tecnica MOSTRE:**

1. **M**ostre a peca (tire da arara, estenda)
2. **O**bserve a reacao do cliente
3. **S**ugira combinacoes ("Essa bermuda fica otima com essa camiseta aqui")
4. **T**oque o tecido, deixe o cliente sentir
5. **R**essalte o beneficio ("Esse tecido seca super rapido")
6. **E**ncaminhe pro provador se mostrar interesse

**Exemplo pratico:**

> "Essa camiseta aqui (tira da arara) e de algodao peruano, bem macia (cliente toca). Olha como o caimento e bom (estende). Com essa bermuda de tactel aqui fica otimo pra praia. Quer experimentar?"

### Objecoes Comuns e Respostas

| Objecao | Resposta |
| ------- | -------- |
| "Ta caro" | "Entendo! Mas olha so: esse tecido dura muito mais que os de loja de departamento. No fim, sai mais barato porque nao precisa trocar todo ano." |
| "Vou pensar" | "Claro! Se quiser, posso separar essas pecas pra voce. Qual seu nome?" |
| "So estou olhando" | "Fique a vontade! Se precisar de algum tamanho, me chama." |
| "Nao tenho dinheiro" | "A gente aceita cartao em ate 3x sem juros. Ou PIX se preferir!" |
| "Nao gostei da cor" | "Temos outras cores! Deixa eu ver o que temos em estoque..." |

### Fechamento da Venda

**Sinais de que o cliente quer comprar:**

- Pergunta sobre pagamento
- Olha a etiqueta varias vezes
- Experimenta mais de uma vez
- Pergunta "vem mais tamanhos?"

**Frases para fechar:**

| Situacao | Frase |
| -------- | ----- |
| Cliente decidiu | "Perfeito! Vai querer que embrulhe pra presente?" |
| Cliente indeciso entre 2 | "Os dois ficaram bons! Levando os dois, te faco um descontinho." |
| Cliente so quer 1 | "Otima escolha! Vai pagar como? Cartao ou PIX?" |

### Pos-Venda

Depois que o cliente pagou:

1. Agradeca: "Muito obrigado pela compra!"
2. Convide a voltar: "Temos novidades toda semana, aparece!"
3. Informe sobre troca: "Se precisar trocar, tem 7 dias, so trazer com etiqueta."

## 5. Processos Operacionais

### Abertura da Loja

**Horario:** 30 minutos antes de abrir

**Checklist:**

- [ ] Ligar luzes e ar condicionado
- [ ] Verificar troco no caixa (minimo R$ 200 em notas pequenas)
- [ ] Conferir se maquininha esta carregada e conectada
- [ ] Organizar araras (pecas no lugar, tamanhos separados)
- [ ] Verificar provadores (limpos, espelhos ok)
- [ ] Arrumar vitrine se necessario
- [ ] Verificar sacolas e embalagens

### Organizacao das Araras

**Como organizar:**

```text
ARARA 1 - CAMISETAS
[P] [M] [G] [GG] [XG]
 |   |   |   |    |
Cores claras --> Cores escuras

ARARA 2 - BERMUDAS
[P] [M] [G] [GG] [XG]
 |   |   |   |    |
Por tipo: Tactel | Moletom | Jeans
```

**Regras:**

- Tamanhos da esquerda (menor) para direita (maior)
- Cores claras na frente, escuras atras
- Pecas sempre no cabide, nao amontoadas
- Repor quando ficar com menos de 3 unidades de um tamanho

### Reposicao de Estoque

**Quando repor:**

- Quando vender uma peca, repor na proxima "pausa" (sem cliente)
- Minimo 3 unidades de cada tamanho visivel

**De onde pegar:**

- Estoque fica no deposito (parte de tras)
- Se nao achar, anotar para pedir ao gerente

**Como registrar falta:**

> Anotar: "DATA - PRODUTO - TAMANHO - FALTANDO"
> Exemplo: "25/01 - Camiseta preta basica - G - Zerou"

### Fechamento de Caixa

**Passo a passo:**

1. **Contar vendas no sistema** - Total de vendas do dia
2. **Contar dinheiro fisico** - Notas e moedas
3. **Conferir cartoes** - Total no sistema da maquininha
4. **Calcular:**

```text
DINHEIRO EM CAIXA (hoje)
- TROCO INICIAL (R$ 200)
= VENDAS EM DINHEIRO

VENDAS EM DINHEIRO + VENDAS CARTAO/PIX = TOTAL DO DIA

TOTAL DO DIA deve bater com SISTEMA
```

5. **Se der diferenca:**
   - Ate R$ 10: Anotar e seguir
   - Mais de R$ 10: Avisar gerente imediatamente

### Trocas e Devolucoes

**Politica:**

| Situacao | Prazo | Condicao | O que fazer |
| -------- | ----- | -------- | ----------- |
| Troca por tamanho | 7 dias | Com etiqueta, sem uso | Trocar na hora |
| Troca por defeito | 30 dias | Qualquer condicao | Trocar na hora |
| Devolucao | 7 dias | Com etiqueta, sem uso | Chamar gerente |
| Devolucao por defeito | 30 dias | Qualquer condicao | Chamar gerente |

**Como fazer troca:**

1. Verificar se esta dentro do prazo (ver cupom)
2. Conferir condicao da peca (etiqueta, sem manchas/cheiro)
3. Pegar nova peca do estoque
4. Registrar troca no sistema
5. Dar novo cupom ao cliente

**Frase para troca:**

> "Sem problema! Vamos trocar agora. Qual tamanho voce precisa?"

## 6. Situacoes Especiais

### Cliente Insatisfeito

**Regra de ouro:** Nunca discuta. Escute primeiro.

**Passo a passo:**

1. **Escute** - Deixe o cliente falar tudo sem interromper
2. **Entenda** - "Entendo sua frustracao..."
3. **Desculpe** - "Sinto muito por isso"
4. **Resolva** - "Vamos resolver agora. O que posso fazer?"
5. **Acompanhe** - Fique com o cliente ate resolver

**Se nao conseguir resolver:** Chame o gerente.

**Frases uteis:**

| Situacao | Frase |
| -------- | ----- |
| Cliente nervoso | "Entendo sua frustracao. Vamos resolver isso juntos." |
| Cliente gritando | "Vou chamar meu gerente para te ajudar da melhor forma." |
| Cliente com razao | "Voce tem toda razao. Desculpa pelo transtorno. Como posso consertar?" |

### Produto com Defeito

**Se cliente reclamar de defeito:**

1. Peca desculpas: "Sinto muito por isso!"
2. Verifique o defeito (real ou mal uso?)
3. Se for defeito de fabrica: troque na hora
4. Se nao tiver em estoque: oferecer vale-troca ou devolucao do dinheiro (chamar gerente)

**Defeito de fabrica x Mal uso:**

| Defeito de Fabrica | Mal Uso |
| ------------------ | ------- |
| Costura soltando | Rasgado por puxao |
| Mancha de tinta | Mancha de comida |
| Tecido desbotou na 1a lavagem | Desbotou apos muitas lavagens |
| Botao faltando | Botao arrancado |

**Na duvida:** Troque. E melhor perder uma venda do que perder um cliente.

### Falta de Troco

**Prevencao:**

- Sempre comecar o dia com R$ 200 em notas pequenas
- Durante o dia, guardar notas pequenas recebidas

**Se acontecer:**

1. Oferecer PIX: "Posso te passar meu PIX? Assim voce nao precisa esperar troco."
2. Perguntar se tem trocado: "Voce nao teria uma nota menor?"
3. Ultimo caso: "Me da 5 minutinhos que vou trocar aqui do lado"

### Emergencias

| Emergencia | O Que Fazer |
| ---------- | ----------- |
| Furto | Nao confronte. Anote descricao. Avise gerente. |
| Queda de energia | Acalme clientes. Abra portas. Nao faca vendas em dinheiro sem sistema. |
| Cliente passa mal | Ofereca agua e cadeira. Se grave, ligue 192 (SAMU). |
| Incendio | Evacue clientes. Ligue 193 (Bombeiros). |
| Assalto | Nao reaja. De o dinheiro. Memorize descricao. Avise policia depois (190). |

**Numero de emergencia:**

- SAMU: 192
- Bombeiros: 193
- Policia: 190

## 7. Metas e Indicadores

### Meta Diaria

| Metrica | Meta | Como Calcular |
| ------- | ---- | ------------- |
| Faturamento | R$ 500 | Total de vendas do dia |
| Pecas vendidas | 15 | Contar no sistema |
| Ticket medio | R$ 80 | Faturamento / Numero de vendas |

**Exemplo:**

```text
Vendas do dia: R$ 600
Numero de clientes: 8
Ticket medio: R$ 600 / 8 = R$ 75
```

### Meta Semanal

| Metrica | Meta |
| ------- | ---- |
| Faturamento | R$ 3.000 |
| Pecas vendidas | 90 |
| Clientes atendidos | 50 |

### Como Calcular Ticket Medio

```text
TICKET MEDIO = FATURAMENTO TOTAL / NUMERO DE VENDAS

Exemplo:
- Segunda: R$ 400 (5 vendas) = R$ 80
- Terca: R$ 600 (10 vendas) = R$ 60
- Quarta: R$ 500 (6 vendas) = R$ 83

Media da semana: R$ 74
```

**Por que importa?** Ticket medio alto = voce esta vendendo bem (kits, combos, mais pecas por cliente).

### Indicadores de Sucesso

| Indicador | Bom | Excelente |
| --------- | --- | --------- |
| Taxa de conversao | 30% | 50%+ |
| Ticket medio | R$ 70 | R$ 100+ |
| Retorno de cliente | 10% | 20%+ |

**Taxa de conversao:** De cada 10 pessoas que entram, quantas compram?

```text
TAXA = (VENDAS / VISITANTES) x 100

Exemplo: 5 vendas / 20 visitantes = 25%
```

## 8. Dicas de Vendas

### Venda Cruzada

Sempre oferecer mais um produto relacionado:

| Se o cliente comprar | Ofereca |
| -------------------- | ------- |
| Camiseta | Bermuda que combina |
| Bermuda | Camiseta que combina |
| Sunga | Camiseta de praia |
| 1 peca | Kit 3 pecas |

**Frase:**

> "Ja que voce vai levar essa bermuda, olha essa camiseta aqui que combina demais. E se levar as duas, te faco um preco especial."

### Senso de Urgencia

Para ajudar cliente indeciso:

- "Esse tamanho esta acabando, restam so 2"
- "Essa cor e a que mais sai"
- "Semana que vem aumenta o preco dessa linha"

**Importante:** So use se for verdade! Nunca minta pro cliente.

### Elogio Sincero

Quando o cliente experimentar:

- "Ficou muito bom esse tamanho em voce!"
- "Essa cor combinou com voce"
- "O caimento ficou perfeito"

**Cuidado:** Seja sincero. Se nao ficou bom, sugira outro tamanho/modelo.
