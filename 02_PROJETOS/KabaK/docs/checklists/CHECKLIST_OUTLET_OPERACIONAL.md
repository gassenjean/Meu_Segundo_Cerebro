# Checklist Operacional - Outlet KabaK

**Objetivo:** Garantir que o Outlet esteja sempre pronto para operação.

**Contexto:** Loja física em rodovia de alto fluxo, foco em moda masculina e praia, operando como queima de estoque (não poluir marca premium do e-commerce).

---

## Checklist Diário - Abertura

**Horário:** 30 min antes da abertura

### Estrutura

- [ ] Energia elétrica funcionando
- [ ] Ar condicionado ligado
- [ ] Iluminação OK (todas as lâmpadas)
- [ ] Espelhos limpos
- [ ] Provadores limpos e organizados

### Produto

- [ ] Araras organizadas por categoria
- [ ] Etiquetas de preço visíveis
- [ ] Peças dobradas corretamente
- [ ] Tamanhos separados (P, M, G, GG)
- [ ] Kits montados e destacados
- [ ] Estoque de reposição acessível

### Caixa

- [ ] Sistema de pagamento funcionando
- [ ] Maquininha carregada/conectada
- [ ] Troco suficiente (R$ 200 em notas pequenas)
- [ ] Sacolas e embalagens disponíveis
- [ ] Notas fiscais configuradas

### Vitrine

- [ ] Manequins com peças da semana
- [ ] Preços visíveis
- [ ] Sem sujeira ou poeira
- [ ] Iluminação destacada

---

## Checklist Diário - Durante o Expediente

**Frequência:** A cada 2-3 horas ou conforme fluxo

### Reposicao de Produtos

- [ ] Verificar araras e repor peças vendidas
- [ ] Manter tamanhos organizados (P, M, G, GG)
- [ ] Redobrar peças desdobradas
- [ ] Reposicionar pecas nos cabides
- [ ] Verificar se kits estao completos

### Atendimento ao Cliente

- [ ] Cumprimentar cliente em ate 30 segundos
- [ ] Perguntar o que procura (tamanho, estilo, ocasiao)
- [ ] Oferecer ajuda no provador
- [ ] Sugerir pecas complementares (venda cruzada)
- [ ] Informar promocoes ativas
- [ ] Agradecer e convidar para voltar

### Padrao de Atendimento Outlet

```text
1. ACOLHER - "Boa tarde! Seja bem-vindo ao KabaK Outlet!"
2. SONDAR - "Procura algo especifico? Temos promocoes incriveis hoje."
3. APRESENTAR - Mostrar pecas, destacar qualidade e desconto
4. EXPERIMENTAR - Direcionar ao provador, oferecer outros tamanhos
5. FECHAR - "Vai levar? Posso empacotar para presente?"
6. DESPEDIR - "Obrigado! Volte sempre, temos novidades toda semana!"
```

### Registro de Vendas

- [ ] Registrar cada venda no sistema imediatamente
- [ ] Emitir nota/cupom fiscal
- [ ] Oferecer sacola adequada ao tamanho da compra
- [ ] Anotar feedback do cliente (se relevante)

### Manutencao Continua

- [ ] Manter area do caixa limpa
- [ ] Verificar provadores apos cada uso
- [ ] Recolher pecas deixadas fora do lugar
- [ ] Manter entrada da loja livre e convidativa

---

## Checklist Diário - Fechamento

**Horário:** Após último cliente

### Caixa

- [ ] Conferir vendas do dia
- [ ] Fechar caixa (conferir valor físico vs sistema)
- [ ] Registrar discrepâncias (se houver)
- [ ] Guardar dinheiro em local seguro
- [ ] Emitir relatório diário

### Estoque

- [ ] Repor peças vendidas nas araras
- [ ] Marcar produtos em falta
- [ ] Separar peças para conserto/descarte
- [ ] Organizar provadores

### Estrutura

- [ ] Desligar ar condicionado
- [ ] Apagar luzes (exceto segurança)
- [ ] Verificar portas e janelas
- [ ] Ativar alarme (se houver)
- [ ] Desligar equipamentos não essenciais

### Registro

- [ ] Preencher relatório diário
- [ ] Anotar observações relevantes
- [ ] Reportar problemas ao gerente

---

## Checklist Semanal

**Dia:** Segunda-feira (ou primeiro dia útil)

### Estoque

- [ ] Inventário rápido (conferir quantidades vs sistema)
- [ ] Identificar bestsellers da semana
- [ ] Identificar produtos parados
- [ ] Solicitar reposição ao gerente
- [ ] Verificar estoque mínimo

### Limpeza

- [ ] Limpeza profunda do piso
- [ ] Limpar vidros e espelhos
- [ ] Higienizar provadores
- [ ] Limpar balcão e caixa
- [ ] Verificar lixeiras

### Organização

- [ ] Reorganizar araras por cor/estilo
- [ ] Atualizar vitrine (peças novas)
- [ ] Verificar precificação correta
- [ ] Remover peças danificadas

### Relatório

- [ ] Compilar vendas da semana
- [ ] Identificar padrões (horários de pico, etc.)
- [ ] Feedback de clientes
- [ ] Sugestões de melhoria

---

## Checklist Mensal

**Dia:** Último dia útil do mês

### Inventário Completo

- [ ] Contagem física de TODO o estoque
- [ ] Conferir com sistema
- [ ] Identificar perdas/furtos
- [ ] Documentar discrepâncias
- [ ] Atualizar sistema

### Manutenção

- [ ] Verificar ar condicionado (filtros)
- [ ] Checar iluminação (trocar lâmpadas)
- [ ] Inspecionar móveis e araras
- [ ] Verificar sistema de segurança
- [ ] Manutenção preventiva geral

### Financeiro

- [ ] Fechamento do mês
- [ ] Conferir entradas vs sistema
- [ ] Reportar ao financeiro
- [ ] Análise de margem por produto

### Equipe

- [ ] Feedback individual (se houver equipe)
- [ ] Treinamento de reciclagem
- [ ] Avaliar necessidades

## Problemas Comuns e Soluções

| Problema | Solução Imediata | Escalar Para |
| -------- | ---------------- | ------------ |
| Maquininha offline | Usar PIX manual | Suporte técnico |
| Falta de troco | Pedir cliente pagar exato/PIX | Gerente |
| Produto sem etiqueta | Consultar sistema/lista | Gerente |
| Cliente insatisfeito | Ouvir, registrar, oferecer solução | Gerente |
| Ar quebrado | Ventilar, informar clientes | Manutenção |
