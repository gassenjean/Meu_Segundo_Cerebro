# Configurando um Aplicativo Interativo na Unreal Engine

## Introdução

Neste guia, vamos explorar como configurar um aplicativo interativo usando a Unreal Engine, com foco em criar uma experiência em tela vertical para um totem interativo. Abordaremos desde o posicionamento da câmera até a programação visual usando Blueprints, passando por configurações essenciais para garantir que o aplicativo funcione conforme o esperado.

## Configuração Inicial da Cena

### Posicionamento da Câmera

O primeiro passo crucial é posicionar corretamente a câmera na cena:

- A câmera deve estar de frente para o personagem ou objeto principal
- Certifique-se de que a câmera capture o corpo inteiro do personagem (no exemplo, a Vivian)
- Este posicionamento inicial é fundamental para a experiência do usuário

### Preparação para Tela Vertical

```
Objetivo: Configurar o aplicativo para rodar em uma tela vertical, como em um totem interativo.
```

- O aplicativo será executado em uma orientação vertical
- Ideal para uso em ambientes empresariais ou espaços públicos
- Requer configurações específicas na Unreal Engine para garantir a exibição correta

## Trabalhando com Blueprints

### O que são Blueprints?

Blueprints são uma ferramenta poderosa na Unreal Engine que permitem programação visual:

- Funcionam como classes e métodos acessíveis visualmente
- Similar a outras ferramentas que usam nós para conexões lógicas (ex: DaVinci Resolve para color grading)
- Permitem interligar classes e métodos através de conexões visuais

### Acessando e Configurando Blueprints

Para começar a trabalhar com Blueprints:

1. Localize o mapa principal do seu projeto
2. Clique nos três pontos (...) ao lado do nome do mapa
3. Selecione "Open Level Blueprint"

Dentro do Level Blueprint, você encontrará dois eventos principais:

- **Event Begin Play**: Executa quando o aplicativo é aberto
- **Event Tick**: Executa continuamente enquanto o aplicativo está rodando

## Configurando a Resolução da Tela

Para configurar a resolução correta para uma tela vertical:

1. A partir do Event Begin Play, adicione o nó "Get Game User Settings"
2. Conecte um nó "Set Screen Resolution"
3. Configure a resolução para 1920x1080 (invertida para orientação vertical):
   - X: 1920
   - Y: 1080
4. Aplique as configurações usando o nó "Apply"

```
Dica: Use a função "Split Struct Pin" no pino de Resolution para definir X e Y separadamente.
```

## Configurando a Câmera Inicial

Para garantir que a cena comece com a câmera na posição correta:

1. Adicione o nó "Set View Target with Blend" após as configurações de resolução
2. Use "Get Player Controller" para definir o Target
3. Arraste a câmera da cena principal para o Blueprint
4. Conecte a referência da câmera ao "New View Target"

### Alternativa para Referência da Câmera

Se preferir, você pode:

- Criar uma variável do tipo "Camera Actor"
- Selecionar a câmera desejada na cena
- Usar essa variável como referência no Blueprint

## Compilação e Teste

Após configurar todos os elementos:

1. Clique em "Compilar" para verificar erros
2. Corrija quaisquer erros que apareçam (preste atenção aos tipos corretos de nós)
3. Salve suas alterações
4. Volte à cena principal
5. Use a opção "Standalone Game" para testar o aplicativo

### Resolução de Problemas Comuns

- Certifique-se de usar "Get Player Controller" e não "Get Player State"
- Verifique se todas as conexões entre os nós estão corretas
- Confirme se a câmera está corretamente referenciada no Blueprint

## Conclusão

Configurar um aplicativo interativo na Unreal Engine para uma tela vertical requer atenção aos detalhes, desde o posicionamento da câmera até a programação visual com Blueprints. Seguindo este guia, você terá as bases para criar experiências interativas impressionantes em totens ou displays verticais.

Lembre-se de sempre testar seu aplicativo após cada alteração significativa e de explorar as vastas possibilidades que a Unreal Engine oferece para criar experiências únicas e envolventes.
