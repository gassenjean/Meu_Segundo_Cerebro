# Integrando Avatares com Inteligência Artificial na Unreal Engine

## Introdução

Este guia aborda o processo de integração de avatares com inteligência artificial na Unreal Engine, utilizando o plugin Convai e o sistema MetaHumans. Vamos explorar como criar um personagem interativo, configurar sua aparência e comportamento, e implementá-lo em um ambiente 3D realista.

## Configuração Inicial do Personagem

### Preparando o Avatar MetaHuman

Para começar, é necessário configurar um avatar MetaHuman na Unreal Engine:

- Acesse o MetaHumans na Unreal Engine
- Selecione um personagem (neste exemplo, usamos "Vivian")
- Abra o Blueprint do personagem com dois cliques

### Integrando com o Convai

Para integrar o avatar com o sistema Convai:

1. Em Class Settings, defina o Parent Class como "Convai Base Character"
2. Adicione o componente "Convai Face Sync"
3. Configure as classes de animação:
   - Para o corpo (Body): selecione "Convai Metalman Body Anim"
   - Para o rosto (Face): escolha "Convai Metalman Face Anim"

4. Compile e salve as alterações

```
Lembre-se: Após adicionar o Convai, a posição do personagem pode mudar. Ajuste conforme necessário para manter o personagem visível e bem iluminado na cena.
```

## Configuração do Character ID

Para vincular o avatar ao personagem criado no Convai:

1. Selecione o avatar na cena
2. Localize o campo "Char ID" nos detalhes do personagem
3. Copie o Character ID do seu personagem no Convai
4. Cole o ID no campo "Char ID" na Unreal Engine
5. Salve o projeto

**Importante:** O avatar usado na Unreal Engine é diferente do avatar exibido no site do Convai. O MetaHuman será usado no ambiente 3D, enquanto o avatar do Convai é para visualização web.

## Criação do Player e Configurações de Jogo

### Adicionando um Player Pawn

Para criar uma perspectiva de jogador:

1. Crie um novo Blueprint do tipo Pawn
2. Nomeie-o (ex: "ConvaiPawn")
3. Em Class Settings, defina o Parent Class como "Convai Base Character"
4. Compile e salve

### Configurando o Game Mode

Ajuste as configurações do jogo:

1. Em World Settings, selecione "Game Mode Override"
2. Escolha "Convai Demo GM"
3. Em Default Pawn, selecione o Pawn criado anteriormente (ex: "ConvaiPawn")

## Testando a Integração

Para testar o avatar integrado:

1. Inicie o jogo em modo Standalone
2. Aguarde a compilação dos shaders e carregamento da cena
3. Use a tecla T para ativar o modo de fala ou Enter para digitar
4. Interaja com o avatar usando voz ou texto

```
Dica: Se o carregamento estiver lento, minimize a Unreal Engine para acelerar o processo de compilação.
```

## Aprimorando a Experiência Visual

### Configurando a Câmera

Para melhorar a visualização do avatar:

1. Adicione uma Cine Camera Actor à cena
2. Posicione a câmera adequadamente em relação ao avatar
3. Ajuste as configurações de Film Back para uma perspectiva vertical (ex: 23.76)
4. Configure o foco da câmera:
   - Use "Manual Focus" ou "Track Focus"
   - Ajuste a distância focal para manter o avatar nítido

### Iluminação e Ambiente

- Ajuste a iluminação da cena para realçar o avatar
- Considere adicionar elementos de cenário para criar um ambiente mais imersivo

## Personalizando o Avatar

### Modificando o Prompt e Comportamento

Para ajustar a personalidade e conhecimentos do avatar:

1. Acesse o painel do Convai
2. Edite o prompt do personagem
3. Salve as alterações e atualize o Character ID na Unreal Engine

### Alterando a Voz

O Convai permite integração com vozes profissionais:

1. Integre o Convai com a API do Eleven Labs
2. Selecione uma voz disponível no Eleven Labs
3. Aplique a nova voz ao seu personagem no Convai

**Observação:** A integração com Eleven Labs pode aumentar ligeiramente o tempo de resposta do avatar.

## Conclusão

A integração de avatares com IA na Unreal Engine oferece possibilidades incríveis para criar experiências interativas e imersivas. Com o uso do Convai e MetaHumans, é possível desenvolver personagens realistas e inteligentes para jogos, simulações e aplicações diversas. Experimente diferentes configurações e personalizações para criar a experiência perfeita para o seu projeto.
