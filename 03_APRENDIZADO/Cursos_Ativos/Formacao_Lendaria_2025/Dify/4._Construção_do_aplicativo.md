# Guia Completo para Criar Aplicativos no DeFi

## Introdução

Este guia abrange os principais aspectos da criação de aplicativos usando a plataforma DeFi. Vamos explorar em detalhes como configurar prompts, utilizar variáveis, trabalhar com bancos de dados e ajustar parâmetros para criar assistentes de IA personalizados e eficientes.

## Construindo o Prompt do Seu Aplicativo

### Modos Básico e Especialista

O DeFi oferece duas opções principais para construir o prompt do seu aplicativo:

- **Modo Básico**:
  - Oferece maior autonomia ao DeFi
  - Ideal para soluções mais simples e rápidas
  - O DeFi auxilia no comportamento do aplicativo
- **Modo Especialista**:
  - Permite maior controle sobre o sistema do aplicativo
  - Possibilita conduzir de forma específica o comportamento e estrutura de resposta
  - Permite guiar o acesso a arquivos e banco de dados
  - Possibilita estruturar passos para o comportamento da resposta

```
Importante: O modo especialista requer mais conhecimento técnico, mas oferece maior customização. O modo básico é recomendado para iniciantes e já oferece resultados satisfatórios na maioria dos casos.
```

### Dicas para Construção de Prompts

- Escreva prompts em inglês para economizar tokens
- Utilize o Markdown para formatar o texto do prompt
- Considere o propósito do seu projeto ao escolher entre modo básico e especialista
- Para insights, consulte modelos de IA externos sobre a construção do seu projeto

## Utilizando Variáveis no Seu Aplicativo

As variáveis permitem coletar informações do usuário para personalizar as respostas. Existem três tipos principais:

1. **Texto curto**:
   - Ideal para nomes, idades, etc.
   - Exemplo: `{nome}`

2. **Parágrafo**:
   - Para respostas mais elaboradas
   - Exemplo: `{objetivo}`

3. **Caixa de seleção**:
   - Para opções predefinidas
   - Exemplo: `{experiencia}` com opções "Sim" e "Não"

### Como Implementar Variáveis

1. Adicione variáveis na seção específica
2. Insira as variáveis no prompt usando chaves: `{nome_da_variavel}`
3. Use condicionais no prompt para personalizar respostas baseadas nas variáveis

Exemplo:

```
Você será um mentor em inteligência artificial. Se referir ao usuário pelo {nome}.Caso a resposta do usuário sobre {experiencia} seja "sim", pode abordar termos mais técnicos sobre IA. Caso seja "não", utilize uma linguagem mais simples.
```

## Recursos Adicionais do DeFi

### Perguntas de Acompanhamento

- O DeFi gera automaticamente opções de perguntas para dar continuidade à conversa

### Conversão de Texto para Áudio

- Possibilidade de converter respostas do chatbot em áudio

### Entrada de Voz

- Permite que usuários façam perguntas por voz

### Citações e Atribuições

- Opção de incluir citações de fontes do banco de dados nas respostas

### Moderação de Conteúdo

- Possibilidade de filtrar palavras específicas

### Anotações Automáticas

- Configuração para armazenar anotações relevantes automaticamente

### Mensagens de Abertura

- Personalização da mensagem inicial e sugestões de perguntas para iniciar o diálogo

## Configurando Modelos e Parâmetros

### Seleção de Modelo de IA

- Escolha o modelo de IA adequado para seu projeto (ex: GPT-3.5, GPT-4)

### Ajuste de Parâmetros

- **Temperatura**: Controla a criatividade das respostas
- **Tokens**: Define o limite de tokens para respostas
- **Presence Penalty e Frequency Penalty**: Ajustam a diversidade do texto
- **Top P**: Influencia a aleatoriedade das respostas

```
O DeFi oferece presets para diferentes estilos de resposta:- Criativo- Balanceado- Preciso
```

## Testando e Refinando Seu Aplicativo

1. Use a seção de teste para interagir com seu aplicativo
2. Analise as respostas geradas e os tokens utilizados
3. Faça ajustes no prompt, variáveis ou parâmetros conforme necessário
4. Lembre-se de resetar a conversa após alterações significativas

## Conclusão

Criar um aplicativo eficiente no DeFi envolve dominar a construção de prompts, o uso inteligente de variáveis e a configuração adequada de parâmetros. Com prática e experimentação, você poderá criar assistentes de IA poderosos e personalizados para diversas aplicações. Lembre-se de sempre considerar o propósito do seu projeto ao fazer escolhas de configuração e design.
