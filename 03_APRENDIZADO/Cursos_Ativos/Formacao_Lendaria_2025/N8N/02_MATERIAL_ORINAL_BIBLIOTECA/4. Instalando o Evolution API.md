## Guia Completo para Instalação e Configuração do Evolution API

## Introdução

Bem-vindos a este guia abrangente sobre a instalação e configuração do Evolution API. Neste ebook, vamos explorar passo a passo como configurar e implantar o Evolution API, uma ferramenta poderosa para automação e integração. Vamos começar entendendo o que é o Evolution API e por que ele é fundamental para o seu fluxo de trabalho.

## Capítulo 1: Preparação do Ambiente

Antes de iniciar a instalação, é essencial preparar o ambiente. Vamos explorar como acessar o n8n e entender os conceitos básicos necessários para a instalação do Evolution API.

### Acessando o n8n

O n8n é uma ferramenta de automação de fluxos de trabalho que integra perfeitamente com o Evolution API. Para acessar o n8n, siga os seguintes passos:

1. **Acessando via Easy Panel**: No seu painel Easy Panel, clique no link fornecido para acessar diretamente o editor do n8n.
2. **URL Direta**: Caso prefira, você pode acessar o n8n digitando a URL correspondente ao seu domínio. Por exemplo, se o seu domínio for `n8n.lendario.online`, a URL será `n8n.editor.n8n.lendario.online`.

### Entendendo o Easy Panel

O Easy Panel é sua central de controle para gerenciar sua VPS (Virtual Private Server). Aqui, você pode criar novos projetos, configurar serviços e monitorar o status de suas aplicações.

## Capítulo 2: Instalação do Evolution API

Agora que estamos preparados, vamos mergulhar na instalação do Evolution API. Este capítulo guiará você através de cada etapa, desde a criação do projeto até a configuração final.

### Criando um Novo Projeto

1. **Acessando o Easy Panel**: Volte à área inicial do seu Easy Panel.
2. **Criando o Projeto**: Clique em "Criar Novo Projeto" e nomeie-o de "Evolution API". Evite usar caracteres especiais para evitar problemas.

### Configurando o Esquema

1. **Copiando o Esquema**: Copie o esquema fornecido e cole no seu projeto.
2. **Criando o Esquema**: Vá até a seção de modelos e clique em "Criar Esquema". Cole o esquema e clique em "Criar".

### Configurando as Portas

1. **Copiando a Porta Interna**: Acesse as credenciais do serviço e copie a porta interna.
2. **Expondo a Porta**: Cole a porta interna na seção "Exponha a Porta" e salve as alterações.

## Capítulo 3: Configuração e Implantação

Agora que o Evolution API está instalado, vamos configurar e implantar a aplicação.

### Configurando o Ambiente

1. **Substituindo a URL do Banco de Dados**: Acesse a seção de ambientes e substitua a URL do banco de dados pela URL interna do seu PostgreSQL.
2. **Configurando a API Key**: Na linha 106 do seu esquema, substitua a API Key pela nova gerada pelo Handelkin Generator.

### Implantando a Aplicação

1. **Forçando a Reconstrução**: Clique em "Forçar Reconstrução" para iniciar o processo de implantação.
2. **Aguardando a Implantação**: Aguarde até que a aplicação termine de ser implantada. Você pode monitorar o status na seção de visão geral.

## Conclusão

Parabéns! Você concluiu a instalação e configuração do Evolution API. Com este guia, você agora tem uma ferramenta poderosa para automação e integração, pronta para ser utilizada em seus projetos. Lembre-se de que a prática leva à perfeição, então não hesite em explorar e experimentar novas configurações.
