# Extração Inteligente de Dados com Modelos de Perguntas e Respostas

## Introdução

Neste ebook, exploraremos uma técnica poderosa para extrair informações valiosas de textos usando modelos de perguntas e respostas baseados em inteligência artificial. Aprenderemos como aplicar essa abordagem para identificar e extrair dados específicos, como endereços de e-mail, de conversas e outros contextos textuais. Esta técnica tem diversas aplicações práticas, especialmente para profissionais de marketing e vendas que desejam obter insights de leads a partir de interações não estruturadas.

## Visão Geral da Técnica

A técnica principal discutida neste ebook envolve o uso de um modelo de IA treinado para responder perguntas com base em um contexto fornecido. Especificamente, usaremos esta abordagem para extrair endereços de e-mail de textos, mas o conceito pode ser expandido para outros tipos de dados.

### Principais componentes:

- Um modelo de IA capaz de responder perguntas
- Um conjunto de dados de texto (contexto) para análise
- Uma pergunta específica para extrair a informação desejada
- Código Python para processar os dados e interagir com o modelo

### Benefícios:

- Extração automatizada de informações específicas
- Capacidade de processar grandes volumes de texto
- Flexibilidade para extrair diferentes tipos de dados

## Preparação do Ambiente e Dados

Para implementar esta técnica, precisamos preparar nosso ambiente de desenvolvimento e os dados que serão analisados.

### Bibliotecas necessárias:

```python
import requestsimport jsonimport pandas as pdfrom tqdm import tqdmimport time
```

- **requests**: Para fazer chamadas à API do modelo de IA
- **json**: Para processar respostas da API
- **pandas**: Para manipulação de dados em formato tabular
- **tqdm**: Para exibir barras de progresso durante o processamento
- **time**: Para adicionar pausas entre as requisições à API

### Preparação dos dados:

1. Crie um arquivo Excel (.xlsx) com uma coluna chamada "contexto"
2. Preencha esta coluna com os textos que deseja analisar
3. Certifique-se de que o arquivo está na mesma pasta do seu script Python

**Exemplo de estrutura do arquivo Excel:**

| contexto ||----------|| Caso tenha alguma dúvida, não hesite em escrever para joao.santos1990@email.com || Para mais informações, entre em contato através do maria_silva@empresa.com.br |

## Implementação do Código

Vamos agora examinar o código Python necessário para implementar esta técnica de extração de dados.

### Leitura do arquivo Excel:

```python
df = pd.read_excel("exemplo.xlsx")if "contexto" not in df.columns:    raise ValueError("A coluna 'contexto' não foi encontrada no arquivo Excel.")if "resposta" not in df.columns:    df["resposta"] = ""
```

Este trecho de código lê o arquivo Excel, verifica se a coluna "contexto" existe e cria uma coluna "resposta" se ela ainda não existir.

### Configuração da requisição à API:

```python
url = "URL_DA_API_AQUI"headers = {"Authorization": "Bearer SEU_TOKEN_AQUI"}def fazer_requisicao(payload):    response = requests.post(url, headers=headers, json=payload)    if response.status_code != 200:        print(f"Erro na requisição: {response.text}")        return None    return response.json()
```

Esta função configura e realiza a requisição à API do modelo de IA. Certifique-se de substituir "URL_DA_API_AQUI" e "SEU_TOKEN_AQUI" com as informações corretas.

### Processamento das linhas:

```python
for index, row in tqdm(df.iterrows(), total=len(df)):    question = "Qual é o e-mail deste texto?"    context = row["contexto"]        payload = {        "question": question,        "context": context    }        try:        response = fazer_requisicao(payload)        if response and "answer" in response:            df.at[index, "resposta"] = response["answer"]        else:            print("Estrutura da resposta não reconhecida")    except Exception as e:        print(f"Erro ao processar linha {index}: {str(e)}")        time.sleep(1)  # Pausa para evitar limites de requisição
```

Este loop processa cada linha do DataFrame, enviando o contexto para a API e armazenando a resposta.

### Salvando os resultados:

```python
df.to_excel("arquivo_atualizado.xlsx", index=False)print("Processamento concluído com sucesso!")
```

Por fim, salvamos os resultados em um novo arquivo Excel.
