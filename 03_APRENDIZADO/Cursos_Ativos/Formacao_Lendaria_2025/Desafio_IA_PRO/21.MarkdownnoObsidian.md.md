## Títulos

Você pode criar títulos usando diferentes níveis de **cabeçalho**. Para isso, utilize um ou mais símbolos "#" no início da linha, seguidos pelo texto do título. Por exemplo:

```
# Título nível 1
## Título nível 2
### Título nível 3
```

Exemplo prático:

# Título nível 1

## Título nível 2

### Título nível 3

## Ênfases no texto

Para enfatizar palavras ou frases no texto, você pode usar negrito, itálico ou texto tachado. Veja alguns exemplos:

```
**Este é um texto em negrito**
*Este é um texto em itálico*
~~Este texto está tachado~~
```

Exemplo prático:

**Este é um texto em negrito**

_Este é um texto em itálico_

~~Este texto está tachado~~

## Citações em bloco

Você pode criar citações em bloco usando o símbolo ">" no início da linha. É possível aninhar citações em bloco adicionando mais símbolos ">". Por exemplo:

```
> Esta é uma citação em bloco.
>> Esta é uma citação aninhada.
```

Exemplo prático:

> Esta é uma citação em bloco.
>
> > Esta é uma citação aninhada.

## Listas

Você pode criar listas ordenadas e não ordenadas no Markdown. Para criar uma lista não ordenada, utilize os caracteres "+", "-", ou "\*". Para criar uma lista ordenada, utilize números seguidos de um ponto ".". É possível criar listas aninhadas adicionando dois espaços antes do início da linha. Veja exemplos:

```
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
    - Subitem 2.2.1
* Item A
* Item B
1. Primeiro item
2. Segundo item
```

Exemplo prático:

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
    - Subitem 2.2.1

* Item A
* Item B

1. Primeiro item
2. Segundo item

## Código

Para destacar trechos de código inline, utilize o caractere de crase "`". Para exibir blocos de código, utilize três crases "```" no início e no final do bloco. É possível especificar a linguagem de programação para obter destaque de sintaxe. Veja exemplos:

````
Código na linha `code`

Bloco de código:

\```
// Exemplo de código em JavaScript
var foo = function(bar) {
  return

 bar++;
};

console.log(foo(5));
\```
````

Exemplo prático:

Código na linha `code`

Bloco de código:

```javascript
// Exemplo de código em JavaScript
var foo = function (bar) {
  return bar++;
};

console.log(foo(5));
```

## Tabelas

Você pode criar tabelas usando o Markdown. Utilize o caractere "|" para separar as colunas e "-" para separar a linha de cabeçalho das demais linhas. Veja um exemplo:

```
| Coluna 1 | Coluna 2 |
| -------- | -------- |
| Dados 1  | Dados 2  |
| Dados 3  | Dados 4  |
```

Exemplo prático:

| Coluna 1 | Coluna 2 |
| -------- | -------- |
| Dados 1  | Dados 2  |
| Dados 3  | Dados 4  |

## Links

Para criar links, utilize a seguinte sintaxe: `[Texto do link](URL)`. Você também pode adicionar um título opcional ao link. Veja exemplos:

```
[Google](https://www.google.com)
[Obsidian](https://obsidian.md "Software de anotações Obsidian")
```

Exemplo prático:

[Google](https://www.google.com)

[Obsidian](https://obsidian.md "Software de anotações Obsidian")

---

## Imagens

Para exibir imagens, utilize a sintaxe: `![Texto alternativo](URL)`. Você também pode adicionar um título opcional à imagem. Veja exemplos:

```
![Minion](https://octodex.github.com/images/minion.png)
![Stormtroopocat](https://octodex.github.com/images/stormtroopocat.jpg "The Stormtroopocat")
```

Exemplo prático:

![Minion](https://octodex.github.com/images/minion.png)

![Stormtroopocat](https://octodex.github.com/images/stormtroopocat.jpg "The Stormtroopocat")
