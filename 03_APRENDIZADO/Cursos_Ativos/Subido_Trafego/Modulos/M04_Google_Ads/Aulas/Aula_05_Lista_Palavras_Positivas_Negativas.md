## INTRODUÇÃO

As palavras-chave em Google Ads têm um poder imenso na hora de fazer anúncios aparecerem para as pessoas certas. Escolher **palavras positivas** (intenção de compra) vs **palavras negativas** (sem intenção) é diferença entre lucro e desperdício. Um clique de quem quer comprar vale ouro. Um clique de quem só quer aprender custa dinheiro.

---

## CONCEITO CENTRAL

**Intenção de Compra vs Curiosidade**

A chave é entender que toda busca comunica uma **intenção silenciosa**. Pessoa procurando "bolo de cenoura fofinho" está pronta. Pessoa procurando "como fazer bolo de cenoura" está aprendendo. Qual você quer pagar?

## GATILHOS DE EXCLUSÃO — Quando Adicionar Negativas

### **1. Análise Semanal de Search Terms Report**

No Google Ads:

1. Vá para **Campaigns → Keywords → Search Terms**
2. Olhe **"Added to report"** (cliques que você pagou)
3. Se 10+ cliques vêm de "como fazer", "tutorial", "grátis"... **EXCLUA**
4. Regra: Se CTR < 1%, investigar. Se conversão = 0%, excluir.

### **2. Padrões a Observar**

| Sinal                                               | Ação                                        |
| --------------------------------------------------- | ------------------------------------------- |
| Muitos cliques em "template" / "modelo"             | Adicionar "template", "modelo" em negativas |
| Cliques em "legging para homem" mas vende só mulher | Adicionar "-homem", "-masculino"            |
| Cliques de cidades fora do raio                     | Usar geolocalização + negativas por estado  |
| High CTR baixa conversão = educação                 | Revisar keywords, adicionar negativas info  |
