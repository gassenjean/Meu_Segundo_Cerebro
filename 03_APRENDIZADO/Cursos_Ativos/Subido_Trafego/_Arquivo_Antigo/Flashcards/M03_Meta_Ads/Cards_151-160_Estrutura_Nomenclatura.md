# FLASHCARDS M03 A04 - ESTRUTURA E NOMENCLATURA

## Card 151

**FRENTE:** Quais são os 3 níveis hierárquicos da estrutura Meta Ads e suas funções?

**VERSO:**

1. **CAMPANHA** - Por quê anunciar + quanto gastar (objetivo + orçamento)
2. **GRUPO DE ANÚNCIOS** - Para quem aparecer + onde aparecer (público + posicionamento)
3. **ANÚNCIO** - O que mostrar (creative + copy + CTA)

Hierarquia: Campanha > Grupo > Anúncio (caixas organizadoras)

---

## Card 152

**FRENTE:** O que significa uma estrutura de campanha "1:5:6" na Meta?

**VERSO:**

- **1** campanha
- **5** grupos de anúncios dentro da campanha
- **6** anúncios dentro de cada grupo

**Anúncios podem ser:** Iguais em todos os grupos OU diferentes por grupo
**Via de regra:** Mesmas 6 variações replicadas em todos os grupos

## Card 153

**FRENTE:** Qual o padrão de nomenclatura sugerido para CAMPANHAS?

**VERSO:**

```
[OBJETIVO] [PÚBLICO F/M/Q] [POSICIONAMENTO] [TAG] Descrição
```

**Exemplo prático:**

```
[CONVERSÃO] [F] [STORIES] [DESAFIO] Vídeo convite
```

**Elementos:**

- OBJETIVO = Por que anunciar
- F/M/Q = Frio/Morno/Quente
- POSICIONAMENTO = Feed/Stories/Reels
- TAG = Identificação específica

## Card 154

**FRENTE:** Como deve ser a nomenclatura dos GRUPOS DE ANÚNCIOS?

**VERSO:**

```
(Nº CRIAÇÃO) - [POSICIONAMENTO] - Nome do público
```

**Exemplo prático:**

```
01 - [FEED] - Envolvimento IG 14D
```

**Elementos:**

- Número 2 dígitos = Ordem de criação (01, 02, 03...)
- [POSICIONAMENTO] = AUTO, FEED, STORIES
- Nome público = Descrição clara da audiência

---

## Card 155

**FRENTE:** Qual padrão para nomear ANÚNCIOS individuais?

**VERSO:**

```
AD(Nº) - Descrição identificável
```

**Exemplo prático:**

```
AD01 - Convite de cabeça para baixo
```

**Regras:**

- AD + número sequencial
- Descrição que permita identificar 6 meses depois
- Evitar abreviações confusas
- Sem caracteres especiais

## Card 156

**FRENTE:** Por que "nomenclatura traz ROI" segundo Pedro Sobral?

**VERSO:**
**Organização = Velocidade de análise = Mais dinheiro**

**Benefícios diretos:**

- Extrair dados mais rápido
- Filtrar campanhas com agilidade
- Encontrar elementos específicos em segundos
- Análise comparativa eficiente
- Reduzir erros operacionais

**ROI real:** Tempo economizado + decisões mais rápidas

## Card 157

**FRENTE:** O que acontece se você copiar campanha/grupo/anúncio e deixar "-cópia" no nome?

**VERSO:**
**PROBLEMA:** Polui o gerenciador e dificulta organização

**Por que remover:**

- Confunde visualmente a conta
- Dificulta filtros e buscas
- Aparenta amadorismo
- Atrapalha análise de dados
- Multiplica confusão com escala

**Regra:** SEMPRE renomear após copiar/colar

## Card 158

**FRENTE:** Qual a diferença funcional entre Campanha e Grupo de Anúncios?

**VERSO:**
**CAMPANHA:**

- Define o OBJETIVO (conversão, engajamento, tráfego)
- Define o ORÇAMENTO total
- Estratégia geral

**GRUPO DE ANÚNCIOS:**

- Define o PÚBLICO-ALVO (para quem)
- Define POSICIONAMENTOS (onde aparecer)
- Pode ter orçamento próprio (exceção)

**Resumo:** Campanha = estratégia | Grupo = execução

---

## Card 159

**FRENTE:** Por que ter um padrão de nomenclatura é obrigatório (não opcional)?

**VERSO:**
**SEM PADRÃO = CAOS GARANTIDO**

**Problemas sem padrão:**

- Impossível encontrar campanhas específicas
- Filtros não funcionam
- Análise comparativa impossível
- Equipe não consegue colaborar
- Escala vira pesadelo

**Com padrão:**

- Conta organizada = mesa de trabalho limpa
- Profissionalismo visível
- Suporte à escalabilidade
