# FLASHCARDS M03 A05 - OBJETIVOS E METAS DESEMPENHO

## **CARD 161**

**FRENTE:** Qual é a função principal do objetivo de RECONHECIMENTO na Meta Ads?
**VERSO:** Martelar uma ideia na cabeça do público-alvo ou anunciar para públicos muito pequenos. Usado para divulgar carrosséis, reforçar mensagens específicas e atingir públicos pequenos completamente.

## **CARD 162**

**FRENTE:** Quais são os 3 micro-objetivos (metas de otimização) do objetivo RECONHECIMENTO?
**VERSO:** 1) **Alcance** - atingir máximo pessoas diferentes; 2) **Impressão** - repetir para mesma pessoa (frequência); 3) **Incrementalidade** - fazer pessoa lembrar do anúncio específico.

## **CARD 163**

**FRENTE:** Quando usar o objetivo de TRÁFEGO na Meta Ads?
**VERSO:** Para enviar pessoas para sites sem pixel/conjunto dados instalados (YouTube, externos), links WhatsApp, blogs, perfis Instagram, ou incentivar ligações em regiões específicas.

## **CARD 164**

**FRENTE:** Diferença entre "Clique no link" vs "Visualizações de página" no objetivo TRÁFEGO?
**VERSO:** **Clique no link:** foco em muitos cliques sem importar carregamento (sem pixel). **Visualizações página:** cliques COM foco no carregamento (requer pixel instalado).

## **CARD 165**

**FRENTE:** Qual é a principal estratégia do objetivo ENGAJAMENTO para transformar públicos?
**VERSO:** Transformar público frio em quente usando método RETINA - divulgar conteúdos para quem não conhece, gerar engajamento, depois vender para essas pessoas aquecidas.

## **CARD 166**

**FRENTE:** Diferença entre Cadastros no SITE vs FORMULÁRIO INSTANTÂNEO?
**VERSO:** **Site:** maior comprometimento, mais informações, leads mais qualificados, maior barreira. **Instantâneo:** preenchimento automático Meta, menos barreira, funciona bem negócios locais.

## **CARD 167**

**FRENTE:** O que é necessário para usar objetivo VENDAS via site de forma eficaz?
**VERSO:** Pixel + Conjunto de dados + API de conversões instalados para medir quando vendas estão acontecendo. Sem isso, Meta não consegue otimizar para conversões.

## **CARD 168**

**FRENTE:** Quando usar VENDAS-CATÁLOGO vs VENDAS-SITE?
**VERSO:** **Site:** vendas diretas via site. **Catálogo:** anúncios dinâmicos baseados ações usuário - Meta pega fotos produtos e divulga para quem já interagiu (ideal e-commerce).

## **CARD 169**

**FRENTE:** Por que objetivo RECONHECIMENTO é "mais fácil" de atingir segundo Pedro?
**VERSO:** Se concretiza apenas com anúncio aparecer para pessoa. Meta avalia probabilidade de "ver anúncio" vs "comprar produto" - ver é sempre mais provável que converter.

## **CARD 170**

**FRENTE:** Para que serve o BOTÃO TURBINAR e quando usar?
**VERSO:** Gerar visitas ao perfil Instagram para pessoas clicarem link bio ou seguirem. Usar para aumentar seguidores Instagram e divulgar perfil como um todo. Sem micro-objetivos - "visitas perfil" já é o objetivo.
