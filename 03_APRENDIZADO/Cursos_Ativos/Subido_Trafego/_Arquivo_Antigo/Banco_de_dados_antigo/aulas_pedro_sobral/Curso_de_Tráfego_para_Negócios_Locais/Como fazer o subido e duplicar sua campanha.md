## [Como fazer o subido e duplicar sua campanha](https://comunidadesobral.s3.amazonaws.com/cst_negocio_local_mfa_a31.pdf)

O documento é um tutorial sobre como duplicar campanhas no Facebook Ads, com um guia passo a passo detalhado. Aqui está um resumo com insights principais:

### Resumo

1. **Introdução**: O objetivo do documento é ensinar como duplicar campanhas no Facebook Ads de forma simples e eficiente.
2. **Passo a Passo**:
   - Acesse o gerenciador de anúncios e clique em “editar” no criativo desejado.
   - Use a opção "duplicar" no menu lateral para criar cópias.
   - Modifique o nome do conjunto de anúncios na cópia para organização.
   - Se a campanha estiver ativa, use a configuração "usar publicação existente" para manter interações sociais (likes, comentários, etc.).
   - Alternativamente, copie o código do anúncio existente e cole na nova publicação.

3. **Soluções para Problemas Comuns**:
   - Caso a publicação existente não seja encontrada, utilize a URL do post original para obter o código.
   - Garanta que as configurações sejam replicadas para manter consistência.

4. **Conceitos Importantes**:
   - Fortaleça campanhas progressivamente sem excluir versões anteriores.
   - Mantenha os "provas sociais" (engajamento) para aumentar a credibilidade do anúncio.

### Insights

- **Eficiência Operacional**: Seguir esse processo otimiza a criação de múltiplos anúncios sem a necessidade de começar do zero.
- **Aproveitamento de Engajamento**: Manter likes e comentários de anúncios anteriores pode aumentar a confiança dos usuários na campanha.
- **Escalabilidade**: A técnica permite a expansão de campanhas em novas geografias ou públicos com rapidez e consistência.
- **Resolução de Problemas**: O documento antecipa dificuldades e oferece soluções práticas, como encontrar e reutilizar publicações específicas.

Esse material é ideal para iniciantes no gerenciamento de Facebook Ads ou para quem busca uma abordagem eficiente para expandir campanhas já existentes.
