## [Eventos padrão, personalizados e conversões personalizadas]()https://comunidadesobral.s3.amazonaws.com/mod_02/cst_m02_a3.1_eventos_padrao_personalizados_e_conversoes_personalizadas.pdf

Esse texto parece ser um guia passo a passo para congurar eventos no Meta Pixel, uma ferramenta de rastreamento de dados criada pela Meta (anteriormente Facebook). O texto é dirigido a gestores de tráfego que desejam configurar eventos no Meta Pixel para rastrear ações dos usuários em seus sites.

Aqui está um resumo dos principais pontos do texto:

1. Após criar e instalar o pixel, é necessário verificar se ele está funcionando corretamente na aba "Visão Geral" do Gerenciador de Anúncios.
2. O Meta Pixel Helper é uma ferramenta que ajuda a verificar se o pixel está configurado corretamente no site.
3. Existem três tipos de eventos que podem ser configurados no Meta Pixel: eventos padrão, eventos personalizados e conversões personalizadas.
4. Os eventos padrão são eventos pré-definidos pelo Meta Ads, enquanto os eventos personalizados podem ser criados pelos gestores de tráfego.
5. As conversões personalizadas permitem que os gestores de tráfego criem eventos personalizados com base em ações específicas dos usuários.
6. Para configurar eventos, é necessário seguir um passo a passo específico, que inclui adicionar eventos, conectar a atividade do site usando um pixel e copiar o código do pixel.
7. Os eventos padrão são recomendados como a melhor opção, seguidos pelos eventos personalizados e, por fim, as conversões personalizadas.

O texto também fornece dicas práticas para configurar eventos no Meta Pixel, como a importância de verificar se as correspondências estão ativadas e de copiar o código do pixel corretamente. Além disso, o texto destaca a importância de priorizar os eventos padrão e de observar os resultados dos dados para ajustar as congurações.
