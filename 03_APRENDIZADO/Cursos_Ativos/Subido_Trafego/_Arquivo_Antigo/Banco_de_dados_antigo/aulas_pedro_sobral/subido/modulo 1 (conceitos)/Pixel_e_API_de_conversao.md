## Pixel

1. levar informações do acontece de dentro do seu site para a fonte de trafego
2. cérebro da sua conta de anúncios
3. criar audiência (publico segmentações) de pessoas que visitaram seu site ou fizeram alguma ação no seu site

## como eu instalo p pixel?

- Manualmente: copiar e colar o código no Head do site.
- Parceiros: verificar se a plataforma de criação de site e um parceiro ou não.
- envio de instruções por e-mail: delegar a instalação do pixel para algum programador.
- GTM (Google Tag Maneger)

## API de conversões

- aumenta o potencial do pixel exercer seus três papeis
- API de conversões fica no servidor

## como o traqueamento pode ser feito?

1. pixel e API de conversões (GTM)
2. Google Analitics + UTM´s (parâmetros de acompanhamentos)
3. paginas especificas para fontes especificas
4. canal de vendas específicos
5. perguntar de onde vc nos conheceu
6. lista das pessoas que compraram (nome, e-mail, telefone, valor da compra.)
7. faturamento na sua ferramenta de recebimento.
