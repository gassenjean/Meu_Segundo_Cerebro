[Lances](https://comunidadesobral.s3.amazonaws.com/mod_01/cst_m01_a8.1_lances.pdf)

O texto apresenta uma explicação sobre como utilizar os lances da forma correta para ganhar os leilões em plataformas de anúncios online, como o Meta Ads e o Google Ads.

Aqui está um resumo dos principais pontos abordados:

1. **O lance pode ser o orçamento ou o valor que você está disposto a pagar por um determinado resultado**.
2. **A estratégia de lance é definir como as fontes de tráfego vão gastar o seu dinheiro**.
3. **No Meta Ads, a opção "volume mais alto" é recomendada**, pois diz para a plataforma encontrar o máximo de resultados com a verba disponível.
4. **No Google Ads, existem várias opções de lances**, incluindo "conversões", "valor da conversão" e "ROAS desejado". É importante definir um valor para o CPA desejado para evitar que o Google gaste mais do que o necessário.
5. **Quanto mais dinheiro você investir, mais você vai aparecer**, mas é importante ganhar os leilões para aparecer mais vezes.
6. **Os testes são fundamentais para determinar os valores dos lances** e ganhar os leilões.

Em resumo, para ganhar os leilões em plataformas de anúncios online, é importante definir uma estratégia de lance eficaz, investir dinheiro suficiente e realizar testes para determinar os valores dos lances.
