## CONCEITOS PRINCIPAIS

### 3 Níveis do Gestor de Tráfego

- **Nível 1 - Apertador:** Foca só no anúncio, CTR, CPC. Resultado inconsistente e reativo.
- **Nível 2 - Otimizador:** Entende sistema de campanha, públicos, orçamentos. Melhor mas ainda tático.
- **Nível 3 - Estratégico:** Domina ecossistema completo, processos científicos replicáveis. Resultado consistente e escalável.

### Template Criação de Campanha Científica

1. **POR QUÊ?** - Qual problema resolver no negócio?
2. **PARA QUEM?** - Que estágio do funil? Que comportamento demonstrou?
3. **O QUÊ?** - Que mensagem precisa ouvir? Que ação tomar?
4. **COMO MEDIR?** - Que métrica indica sucesso? Quando otimizar vs pausar?

### Hierarquia de Decisões

```
SISTEMA COMPLETO (negócio + funis)
    ↓
CAMPANHA (objetivo específico)
    ↓
CONJUNTO (público segmentado)
    ↓
ANÚNCIO (mensagem + criativo)
```

### Shift Mindset: Reativo → Estratégico

- **Antes:** Reagir a métricas ruins, trocar criativo quando CTR baixo
- **Depois:** Analisar padrões científicos, entender POR QUE métrica está ruim
- **Processo:** Review estratégico semanal toda segunda 9h vs otimização diária reativa

## TEMPLATES/CHECKLISTS

### Review Estratégico Semanal (Segunda 9h)

```
1. Métricas da semana: identificar padrões vs anomalias
2. Hipóteses: por que os resultados aconteceram?
3. Decisões: que ajustes fazer baseado em dados?
4. Testes: que hipóteses validar esta semana?
5. Prevenção: que problemas antecipar?
```

### Análise Problema Sistêmico

```
CTR baixo → analisar match produto-público-mensagem
CPC alto → revisar qualidade anúncio + relevância
ROAS caiu → análise funil completo + sazonalidade
Carrinho alto → problema checkout/frete/confiança
```
