[Pixel e API de Conversões](https://comunidadesobral.s3.amazonaws.com/mod_02/cst_m02_a03_pixel_e_api_de_conversoes.pdf)



A introdução menciona que o conteúdo sobre o pixel do Meta Ads e a API de conversões foi abordado anteriormente no módulo 1 do curso. O objetivo desta leitura é detalhar como funciona o pixel do Meta Ads e a API de conversões.

O passo a passo para configurar o pixel e a API de conversões do Meta Ads é apresentado da seguinte forma:

1. Acessar o menu "Fonte de dados" no menu lateral esquerdo para visualizar todos os pixels.
2. Clicar em "Conectar fontes de dados" para criar um novo pixel.
3. Escolher a opção "Web" e clicar em "Conectar" para criar um novo conjunto de dados.
4. Dar um nome ao novo pixel e clicar em "Criar".
5. Verificar a criação do pixel e clicar em "Configurações" no menu superior.
6. Ativar a "Correspondência avançada automática" e todas as opções abaixo.
7. Ativar a opção "Rastrear eventos automaticamente sem código".
8. Voltar ao menu "Visão geral" e escolher uma das duas maneiras de instalar o pixel:
 * Enviar eventos de um servidor através da opção "Configurar a API de conversões".
 * Colete eventos do site das atividades no navegador.
9. Para instalar o pixel, copiar o código e instalá-lo na head do site.
10. Ativar a opção "Ativar correspondência avançada automática".
11. Existem outras opções para instalar o pixel, como procurar parceiros no painel "Conectar a atividade do site usando um pixel".

O texto também menciona que é importante consultar outros materiais complementares dentro do portal de conteúdos e praticar dentro do Gerenciador de Anúncios.
