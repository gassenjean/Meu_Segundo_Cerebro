## [Criação de campanhas](https://comunidadesobral.s3.amazonaws.com/mod_02/cst_m02_a06_criacao_de_campanhas.pdf)



Este texto é uma introdução a um tutorial sobre como criar campanhas no Meta Ads.  Ele utiliza uma abordagem didática, comparando o processo de criação de uma campanha a montar um quebra-cabeça.  O autor, aparentemente chamado Pedro, enfatiza a importância de entender cada componente individual (nome, objetivo, orçamento, etc.) antes de tentar criar uma campanha completa.  Ele promete explicar cada componente em materiais subsequentes e adverte contra pular etapas para evitar dificuldades futuras.  O tom é informal e motivacional, usando expressões como "Tamo junto!" para criar uma conexão com o público.
