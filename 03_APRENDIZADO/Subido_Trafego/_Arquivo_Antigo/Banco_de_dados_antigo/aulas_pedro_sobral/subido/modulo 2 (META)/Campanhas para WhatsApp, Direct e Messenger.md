## [Campanhas para WhatsApp, Direct e Messenger](https://comunidadesobral.s3.amazonaws.com/mod_02/cst_m02_a7.4_campanhas_para_whatsapp_direct_e_messenger.pdf)

O material aborda a criação de campanhas para WhatsApp, Direct e Messenger dentro do Gerenciador de Anúncios da Meta, destacando os seguintes pontos principais:

### **Introdução**

- O objetivo é ensinar como configurar campanhas para as três plataformas de mensagens.
- A configuração é semelhante para todas e utiliza o objetivo de campanha "engajamento".

---

### **Passo a Passo para Criar Campanhas**

1. **Início da Campanha**:
    
    - Acesse o Gerenciador de Anúncios e clique em “+ criar”.
    - Escolha o objetivo “engajamento”.
    - Nomeie a campanha com “[MENSAGEM] [WHATSAPP ou outro app]”.
2. **Definição do Público**:
    
    - Utilize públicos pré-criados via API ou configure manualmente no Gerenciador de Anúncios.
    - Exemplo: segmentar quem visualizou anúncios ou interagiu com a marca.
3. **Tipos de Campanha**:
    
    - **Mensagens Personalizadas (Advantage +)**: Configuração simplificada, mas limitada a um local de anúncio por vez.
    - **Engajamento Manual**: Permite maior personalização e testes antes de usar campanhas automáticas.
4. **Configurações Importantes**:
    
    - Ative o “orçamento de campanha Advantage+”.
    - Configure para "maximizar o número de conversas" nas metas de engajamento.
    - Utilize posicionamento manual, evitando múltiplos aplicativos simultaneamente.

---

### **Detalhes de Configuração**

- **Modelo de Mensagem**:
    
    - Crie mensagens automáticas para facilitar o início da interação.
    - Varie formatos e teste diferentes abordagens para maximizar resultados.
- **Anúncios**:
    
    - Personalize títulos, mídias e chamadas para ação (exemplo: “enviar mensagem”).
    - Inclua rastreamento com “eventos do site”.
- **Hierarquização de Públicos**:
    
    - Use estratégias como excluir quem já interagiu para alcançar novos leads.

---

### **Dicas e Boas Práticas**

- Teste diversos modelos e públicos para entender o que funciona melhor.
- Evite anúncios muito complexos para facilitar a interação do usuário.
- Assista à [[Sua_Mente_Lendária_icloud/Recursos/aulas_pedro_sobral/subido/lives/live 217]] para um passo a passo ainda mais detalhado.

### **Conclusão**

O material reforça a importância de praticar o que foi aprendido e testar diferentes configurações para maximizar os resultados das campanhas.
