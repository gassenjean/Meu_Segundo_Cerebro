## [Posicionamentos](https://comunidadesobral.s3.amazonaws.com/mod_02/cst_m02_a6.10_posicionamentos.pdf)

O documento apresenta um guia sobre posicionamentos no Gerenciador de Anúncios do Meta Ads, com os seguintes pontos principais:

### **Resumo dos Capítulos**

1. **Introdução**:
    
    - Explica o conceito de posicionamento no Meta Ads, que define onde os anúncios aparecem, como Instagram, Facebook, Reels e Stories.
    - Destaca dois tipos de posicionamento: **Advantage+** (recomendado pela plataforma) e **manuais** (21 tipos disponíveis).
      
2. **O que são os posicionamentos?**:
    
    - Detalha o funcionamento dos posicionamentos no Meta Ads.
    - Explica que o Audience Network inclui anúncios em aplicativos e sites parceiros além do Facebook.
      
3. **Como escolher os posicionamentos?**:
    
    - Recomenda o uso do Advantage+ para maximizar resultados, deixando a seleção nas mãos da inteligência do Meta Ads.
    - Orienta sobre quando usar posicionamentos manuais para maior controle.
      
4. **Quando usar posicionamentos manuais?**:
    
    - Indica situações específicas:
        1. Quando houver irregularidades em determinados posicionamentos.
        2. Ao isolar um canal (ex.: focar exclusivamente no Instagram).
        3. Para anúncios altamente específicos.
           
5. **Criativo do anúncio**:
    
    - Fornece um passo a passo para configurar criativos com base nos posicionamentos escolhidos, incluindo:
        - Ajustar dispositivos e sistemas operacionais.
        - Definir listas de bloqueio e filtros de inventário.
    - Explica configurações avançadas, como segmentação por versões de sistema operacional.

### **Conclusão**

O documento ressalta a importância de testar e entender as ferramentas para melhor aproveitamento das campanhas. A prática contínua é enfatizada como essencial para obter domínio das configurações e estratégias de anúncios.
