[Escala como aumentar seus gastos.](https://comunidadesobral.s3.amazonaws.com/mod_01/cst_m01_a11_escala_como_aumentar_seus_resultados.pdf)

Esse texto é uma introdução ao conceito de escalar no mercado digital e como utilizar o tráfego pago para alcançar esses objetivos. O autor apresenta três maneiras de escalar produtos no mercado digital:

1. Vender para mais pessoas;
2. Vender mais vezes para as mesmas pessoas;
3. Cobrar mais caro pelos produtos.

Além disso, o autor destaca que o tráfego pago não é uma escada infinita e que existem limitadores do processo de escala, como o leilão, que pode tornar mais caro ganhar os leilões.

O autor também apresenta algumas estratégias para escalar as campanhas de tráfego pago, incluindo:

1. Aumentar o orçamento;
2. Anunciar para novos públicos;
3. Novos anúncios;
4. criar novas fonte de trafego;
5. Explorar novas fontes de tráfego;
6. Melhorar o processo de conversão, incluindo a melhoria das cópias dos anúncios, textos de e-mails marketing, textos da landing page, automações, usabilidade do meu site;

O autor também alerta para a estratégia comum de anunciar para o mesmo público várias vezes, na esperança de alcançar mais pessoas, e enfatiza a importância de entender os conceitos apresentados antes de seguir para as etapas práticas.
