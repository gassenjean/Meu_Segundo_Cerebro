# M03 A10 - Posicionamentos

## Conceito Central
**Onde aparecer = Posicionamento.** Use automático em 99% dos casos, manual só em situações específicas. Formatos múltiplos de criativo desbloqueiam o poder total do automático.

## Insights que Prestam

### 1. Automático (Advantage Plus) Domina
- Meta decide onde aparecer baseado em performance
- Antigamente era ruim, hoje DESTRÓI manual em resultados
- Foi um dos primeiros recursos Advantage que realmente funcionou

### 2. Manual: 4 Situações Específicas
- **Posicionamento morto:** Recurso detalhamento mostra onde não converte
- **Público desqualificado:** Facebook = poder aquisitivo menor que Instagram (Brasil)
- **Garantir gasto:** Isolar para testar ou estratégia específica
- **Anúncio nativo:** Enquete Stories, recursos exclusivos de plataforma

### 3. Formatos Múltiplos = Hack Ignorado
- Quadrado (feed), vertical (stories/reels), retangular (desktop)
- Meta pede diferentes formatos na criação
- Cada formato libera posicionamentos específicos
- **Erro comum:** Usar só um formato e limitar alcance

### 4. Detalhamento: Ferramenta Esquecida
- Mostra exatamente onde gastou dinheiro
- Identifica posicionamentos vampiros (gastam sem converter)
- Base para decisões de exclusão manual

### 5. Wi-Fi/Sistema Operacional: Morto
- Recurso do passado quando mobile era ruim
- Hoje irrelevante: 4G/5G rápido, celulares bons baratos
- Só menciona para não parecer desatualizado

### 6. Anúncio Camuflado: Ouro Puro
- Enquete interativa Stories = não parece anúncio
- Recursos nativos de plataforma convertem mais
- Requer posicionamento manual específico para liberar

## Aplicação Real

### Para KabaK:

- **Teste isolado:** Stories com enquete "Você usa tamanho M ou G?" 
- **Exclusão:** Audience Network se detalhamento mostrar lixo
- **Instagram focus:** Manual só se dados provarem conversão 2x melhor

## Verdade Nua

Pedro direto ao ponto nessa aula. Sem drama sobre "algoritmo mágico" ou "segredo dos gurus". 

**99% automático** não é exagero - é realidade comprovada. Manual só faz sentido em casos muito específicos.

**Enquete interativa** é exemplo perfeito de pensar fora da caixa. Mas precisa effort extra de criação.
