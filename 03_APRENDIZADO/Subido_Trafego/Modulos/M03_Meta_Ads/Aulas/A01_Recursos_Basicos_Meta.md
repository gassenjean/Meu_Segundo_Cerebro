# M03 A01 - Recursos Básicos para Criar Anúncios na Meta

## Conceito Central
**Infraestrutura profissional é obrigatória: perfil real + página + Instagram comercial + Business Manager**

## Conceitos Principais

### 1. Perfil Facebook - Obrigação Inegociável
- **Realidade:** Zuckerberg manda, você obedece
- **Requisitos:** Foto real, amigos, interações, grupos
- **Paranoia Meta:** Detecta contas fake anunciando produtos proibidos
- **Sem perfil = sem anúncios** (não tem conversa)

### 2. Hierarquia de Recursos
- **Perfil:** Pessoal (você como pessoa)
- **Página:** Profissional (empresa/marca)
- **Gestor não precisa página própria** - usa a do cliente
- **NUNCA pedir senha** - acesso liberado via email

### 3. Instagram Comercial vs Criador
- **SEMPRE usar Comercial** (não criador de conteúdo)
- **Limitação crítica:** Criador bloqueia segmentações avançadas
- **Exemplo:** Público que interagiu recentemente não funciona
- **Mudança simples mas crucial** para remarketing

### 4. Gerenciador de Anúncios
- **Ambiente de criação:** Campanhas, anúncios, segmentações
- **Gestor recebe acesso** (não precisa ser dono)
- **Metáfora da chave:** Cliente dá acesso sem dar senha
- **Profissionalismo básico** em qualquer operação

### 5. Business Manager (BM)
- **Central de ativos:** Páginas, pixels, domínios, catálogos
- **Obrigatório mesmo para cliente pequeno** (1 página, 1 conta)
- **Sinal de seriedade** para Meta
- **Organização profissional** vs amadorismo

### 6. Mínimo Absoluto vs Profissional
- **Mínimo técnico:** Gerenciador + página/Instagram
- **Mínimo profissional:** Todos os recursos estruturados
- **Meta avalia profissionalismo** pela estrutura
- **Amador faz mínimo, profissional faz certo**

## Aplicação Prática

### Para KabaK
1. **Instagram comercial** configurado (não criador)
2. **Business Manager** organizando todos ativos
3. **Página Facebook** separada do pessoal
4. **Pixels/conjuntos dados** centralizados no BM
5. **Acessos via BM** para qualquer colaborador

### Para Outlet Gabriele
1. **Mesma estrutura profissional** do KabaK
2. **Instagram comercial** para remarketing completo
3. **Catálogos produtos** integrados no BM
4. **Domínio verificado** para iOS 14+
5. **Separação clara** pessoal vs negócio

## Insights Estratégicos

### Realidade do Mercado
- **Meta quer seus dados pessoais** (fingimento do Pedro sobre ser "esquisito")
- **Paranoia com fake news/produtos proibidos** aumentando
- **Profissionalismo = sobrevivência** no ecossistema Meta
- **Gestor amador pede senha** = red flag gigante

### Erros Comuns
1. **Usar criador de conteúdo** no Instagram (limita remarketing)
2. **Pedir senha do cliente** (amadorismo puro)
3. **Não usar Business Manager** (parecer pilantra)
4. **Conta fake no Facebook** (ban inevitável)
5. **Fazer só o mínimo** (sinal de amador)

## Observações Críticas

**Contradição Pedro:** Diz ser "esquisito" precisar de perfil Facebook mas é óbvio que Meta quer vincular identidade real aos anúncios. Marketing de vigilância disfarçado de "segurança".

**Realidade crua:** Ou você joga o jogo da Meta com todas as regras ou fica de fora. Não tem meio termo.

**Profissionalismo theater:** Business Manager para cliente com 1 página é exagero técnico, mas necessário para "parecer sério" aos olhos da Meta.
