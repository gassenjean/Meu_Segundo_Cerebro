## CONCEITOS PRINCIPAIS

**Tag vs Ação de Conversão:**
- Tag do Google Ads = pixel único para TODAS as páginas do site (equivalente pixel Meta)
- Ação de Conversão = evento específico para cada coisa que queremos medir (compra, cadastro, clique)
- 1 Tag por conta, múltiplas ações de conversão por objetivos

**Diferença crítica Google vs Meta:**
- Google: Tag + Ação separadas, lances manuais possíveis
- Meta: Pixel tudo-em-um, lances automáticos prioritários

**Hierarquia ID simples:**
- AW-XXXXXXXXXX = ID da tag (código único da conta)
- /YYYYYYYYYY = ID da ação específica
- Não precisa decorar, só reconhecer padrão

**3 formas instalação:**
1. Google Tag Manager (recomendado - curso CST disponível)
2. Manual (copiar snippet no head/body)
3. Dev (enviar código por email)

**Configurações ação conversão:**
- Categoria (compra, lead, visualização)
- Valor (R$ ou R$0 para cadastro)
- Contagem: Uma (1x usuário) vs Todas (múltiplas por usuário)
- Janela atribuição + modelo atribuição
