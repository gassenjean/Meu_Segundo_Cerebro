# Criando Vídeos Automatizados com CreatorMate

## Introdução

Este ebook explora o uso da ferramenta CreatorMate para criar vídeos automatizados a partir de templates. Veremos como configurar uma template, adicionar elementos dinâmicos e integrar com APIs para gerar vídeos personalizados em larga escala.

## Visão Geral do CreatorMate

O CreatorMate é uma API de integração que permite criar vídeos a partir de templates pré-definidas. Suas principais características são:

- Editor visual para criar sequências de vídeo
- Suporte a elementos dinâmicos (variáveis)
- Integração via API para automação
- Templates personalizáveis

```
O CreatorMate é ideal para gerar vídeos em larga escala de forma automatizada, integrando áudio, imagens e texto dinamicamente.
```

### Vantagens do CreatorMate

- Facilidade de uso com interface visual
- Flexibilidade para personalizar templates
- Automação via API
- Renderização na nuvem (sem necessidade de servidor próprio)
- Diferentes planos conforme volume de uso

### Alternativas

Como alternativa ao CreatorMate, é possível implementar a geração de vídeos manualmente usando Python e FFmpeg. Porém, isso requer:

- Desenvolvimento próprio do código
- Servidor/VPS para processamento
- Maior complexidade técnica

## Criando uma Template no CreatorMate

O processo de criação de uma template envolve os seguintes passos:

1. Fazer login na plataforma
2. Clicar em "New" para criar nova template
3. Escolher um layout base (ex: áudio + subtítulos)
4. Definir formato (ex: 1x1 square para redes sociais)
5. Adicionar e configurar elementos:
    - Imagem de fundo
    - Logo
    - Áudios
    - Textos

### Configurando Elementos Dinâmicos

Para tornar elementos dinâmicos (variáveis):

1. Selecionar o elemento
2. Em "Properties", marcar como "Dynamic"
3. Definir duração como "Alto" para preencher timeline
4. Renomear elementos para identificação (ex: audio_1, audio_2, etc)

```
É fundamental marcar os elementos como dinâmicos e com duração "Alto" para que sejam preenchidos corretamente via API.
```

### Organizando a Timeline

- Arrastar elementos para organizar sequência
- Cuidado para não sobrepor áudios
- Alinhar elementos corretamente

### Configurações Adicionais

- Adicionar animações (ex: spin no logo)
- Ajustar tamanho e posição dos elementos
- Definir cores, fontes, etc.

## Integrando com API

Após configurar a template, é possível integrá-la via API:

1. Clicar em "Use Template"
2. Copiar código de integração
3. Identificar variáveis dinâmicas (ex: audio_1, audio_2, etc)
4. Fazer requisição HTTP POST para URL da API
5. Passar parâmetros:
    - Template ID
    - Valores para variáveis dinâmicas
    - Chave de API

### Exemplo de Integração

```python
import requestsurl = "https://api.creatormate.com/v1/render"payload = {  "template_id": "abc123",  "modifications": {    "audio_1": "https://exemplo.com/audio1.mp3",    "audio_2": "https://exemplo.com/audio2.mp3",    "titulo_podcast": "Meu Podcast"  }}headers = {  "Content-Type": "application/json",  "Authorization": "Bearer SUA_CHAVE_API"}response = requests.post(url, json=payload, headers=headers)print(response.json())
```

## Automatizando com Make (Integromat)

Para automatizar o processo de geração de vídeos, podemos usar o Make (antigo Integromat):

1. Criar um cenário no Make
2. Adicionar módulo HTTP "Make a request"
3. Configurar requisição POST para API do CreatorMate
4. Usar agregadores para iterar sobre múltiplos áudios
5. Mapear variáveis dinâmicas da template

### Configurando Requisição HTTP

- URL: Endpoint da API CreatorMate
- Método: POST
- Headers:
    - Content-Type: application/json
    - Authorization: Bearer SUA_CHAVE_API
- Body: JSON com template_id e modifications

### Mapeando Variáveis Dinâmicas

- Usar agregadores para iterar sobre lista de áudios
- Mapear cada áudio para sua respectiva variável (audio_1, audio_2, etc)
- Adicionar título e outros metadados dinâmicos

```
Dica: Use a posição do iterador para mapear dinamicamente o número de áudios, permitindo flexibilidade na quantidade.
```

## Processamento e Entrega do Vídeo

Após enviar a requisição, o CreatorMate inicia o processamento:

1. Vídeo entra na fila de renderização
2. Tempo de processamento varia conforme complexidade/duração
3. Ao finalizar, retorna URL do vídeo gerado

### Próximos Passos

Para finalizar o fluxo:

1. Implementar webhook ou timer para verificar conclusão
2. Fazer download do vídeo gerado
3. Fazer upload para plataforma de distribuição (ex: WhatsApp)
4. Notificar usuário sobre vídeo pronto

## Conclusão

O CreatorMate oferece uma solução poderosa para automatizar a criação de vídeos em larga escala. Ao combinar templates personalizáveis, elementos dinâmicos e integração via API, é possível gerar conteúdo audiovisual de forma eficiente e escalável.

Pontos-chave:

- Configure templates com elementos dinâmicos
- Integre via API para automação
- Use ferramentas como Make para orquestrar o fluxo
- Considere tempo de processamento ao planejar entregas

Com a abordagem correta, é possível criar um pipeline robusto de geração automatizada de vídeos para diversas aplicações.
