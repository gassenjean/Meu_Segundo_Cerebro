# Convertendo Dados do Instagram: De JSON para Excel

## Introdução

Neste ebook, abordaremos um processo prático e valioso para profissionais de marketing digital e analistas de dados: a conversão de mensagens do Instagram do formato JSON para Excel. Este processo faz parte da etapa de transformação no fluxo de ETL (Extração, Transformação e Carregamento) de dados. Aprenderemos como extrair dados do Instagram, processar arquivos JSON e convertê-los em um formato mais amigável para análise.

## Extração de Dados do Instagram

### Acessando Suas Informações

Para iniciar o processo, siga estes passos:

1. Acesse sua conta do Instagram
2. Vá para a aba "Mais"
3. Selecione "Sua atividade"
4. Clique em "Baixar suas informações"
5. Escolha "Continuar"

### Selecionando os Dados

- Marque "Atividade atual" para dados recentes
- Você pode programar extrações regulares para facilitar análises periódicas
- Selecione "Baixar ou transferir informações"
- Escolha sua conta do Instagram

### Personalizando a Extração

- Opte por "Algumas das suas informações" para um controle mais preciso
- Para este exemplo, focaremos em "Mensagens"
- Selecione o intervalo de datas desejado
- Escolha o formato JSON
- Clique em "Criar arquivos"

```
Dica: Extrair mensagens do Instagram pode ser especialmente útil para empresas que desejam analisar interações com clientes ou treinar chatbots para respostas automáticas.
```

## Estrutura dos Dados JSON

Após a extração, você receberá uma pasta contendo subpastas, cada uma com um arquivo JSON. A estrutura típica de um arquivo JSON de mensagem inclui:

- Nome do remetente
- Timestamp (data e hora da mensagem)
- Conteúdo da mensagem
- Informações de geolocalização (se disponíveis)

**Exemplo de estrutura JSON:**

```json
{  "sender_name": "Nome do Remetente",  "timestamp_ms": 1234567890123,  "content": "Conteúdo da mensagem",  "share": {    "link": "https://..."  }}
```

## Processo de Conversão

Para converter os arquivos JSON em um único arquivo Excel, utilizaremos um script Python. Este script realiza as seguintes operações:

1. Lê todos os arquivos JSON em uma pasta específica
2. Processa cada arquivo, extraindo informações relevantes
3. Converte timestamps para formato legível
4. Organiza os dados em um DataFrame do Pandas
5. Exporta o DataFrame para um arquivo Excel

### Requisitos do Script

Para executar o script, você precisará:

- Python instalado em sua máquina
- Biblioteca Pandas instalada (`pip install pandas`)

### Principais Funções do Script

O script inclui várias funções importantes:

- **Carregamento de arquivos JSON**: Lê e valida cada arquivo JSON
- **Correção de codificação**: Assegura que caracteres especiais (como acentos) sejam exibidos corretamente
- **Processamento de dados**: Extrai informações relevantes de cada mensagem
- **Conversão de timestamp**: Transforma o formato de data/hora para um formato legível
- **Renomeação de colunas**: Atribui nomes mais intuitivos às colunas do DataFrame
- **Exportação para Excel**: Salva todos os dados processados em um único arquivo Excel

## Executando o Script

Para usar o script:

1. Certifique-se de ter Python e Pandas instalados
2. Copie o script fornecido
3. Ajuste o caminho da pasta de entrada (onde estão os arquivos JSON)
4. Defina o caminho de saída para o arquivo Excel
5. Execute o script

```python
# Exemplo de como definir os caminhos no scriptroot_folder = "C:/Users/SeuUsuario/Downloads/instagram_data/messages/inbox"output_file = "C:/Users/SeuUsuario/Downloads/mensagens_instagram.xlsx"
```

Após a execução, você terá um arquivo Excel contendo todas as mensagens organizadas em colunas como:

- Nome (remetente)
- Data
- Conversa (conteúdo da mensagem)
- Geolocalização (se disponível)
