# Guia Prático: Cargas de Dados no SupaBase

## Introdução

Neste guia prático, vamos explorar como realizar cargas de dados completas e incrementais no SupaBase, uma plataforma moderna de banco de dados. Este documento é baseado em uma aula prática e fornece instruções passo a passo para criar um projeto, configurar tabelas e importar dados de forma eficiente.

## Criando um Novo Projeto no SupaBase

### Passos Iniciais

1. Acesse sua conta no SupaBase
2. Na tela inicial, clique em "Start Your Project" ou "New Project"
3. Escolha sua organização
4. Dê um nome ao projeto (ex: "Projeto Aula Carga")

### Configurações Importantes

- **Senha do Projeto**:
    
    - Crie uma senha forte e complexa
    - É crucial para a segurança dos seus dados
- **Região do Servidor**:
    
    - Escolha a região mais próxima (ex: Norte da Califórnia)
    - Observe que agora há opções de servidores em outras regiões

```
ATENÇÃO: A escolha da senha e da região do servidor são decisões críticas para a segurança e desempenho do seu projeto.
```

## Preparando os Dados para Importação

### Organização dos Arquivos

- Prepare seus arquivos de dados em formato Excel ou CSV
- Para este exemplo, usaremos dois arquivos:
    1. **Vendas01**: Contém dados dos primeiros 10 clientes
    2. **Vendas02**: Contém dados do 11º ao 20º cliente

### Estrutura dos Dados

- Colunas típicas: ID, Nome do Cliente, Item, Valor, Forma de Pagamento
- Certifique-se de que os dados estão limpos e formatados corretamente

## Realizando uma Carga Completa

### Criando uma Nova Tabela

1. No SupaBase, vá para "Table Editor"
2. Clique em "Create a New Table"
3. Nomeie a tabela (ex: "Venda01")
4. Opcional: Adicione uma descrição

### Importando Dados via Planilha

1. Selecione "Import Data via Spreadsheet"
2. Arraste e solte seu arquivo Excel ou CSV na área designada
3. O SupaBase lerá automaticamente a estrutura dos dados

### Configurando a Tabela

- **Seleção de Colunas**:
    
    - Você pode escolher quais colunas importar
    - Desmarque as colunas que não deseja incluir
- **Configurações de Segurança**:
    
    - Mantenha as configurações de segurança ativadas para proteger seus dados
- **Definindo a Chave Primária**:
    
    - Selecione a coluna que servirá como chave primária (geralmente o ID)
    - Marque a opção "Primary Key" para esta coluna

### Finalizando a Importação

1. Revise todas as configurações
2. Clique em "Save" para criar a tabela e importar os dados
3. Observe a indicação de chave primária (ícone de chave) na coluna selecionada

```
DICA: A definição correta da chave primária é essencial para futuras relações entre tabelas e para a integridade dos dados.
```

## Realizando uma Carga Incremental

### Quando Usar Cargas Incrementais

- Útil quando novos dados surgem e precisam ser adicionados à base existente
- Ideal para atualizações regulares sem recarregar toda a base

### Métodos de Carga Incremental

1. **Inserção Manual**:
    
    - Use "Insert Row" para adicionar linhas individuais
    - Útil para pequenas quantidades de dados
2. **Importação via CSV**:
    
    - Método preferido para grandes volumes de dados
    - Mais eficiente e menos propenso a erros

### Processo de Importação Incremental

1. Na tabela existente, clique em "Insert"
2. Escolha "Import data via CSV"
3. Arraste e solte o novo arquivo (ex: Vendas02)
4. O SupaBase reconhecerá automaticamente a estrutura compatível
5. Revise os dados a serem importados
6. Confirme a importação

### Considerações Importantes

- O SupaBase ignora automaticamente o cabeçalho se for idêntico ao existente
- Certifique-se de que os novos dados são compatíveis com a estrutura da tabela
- Verifique se não há duplicação de chaves primárias

## Considerações Avançadas e Futuras Aplicações

### Limitações do Método Atual

- O método demonstrado é adequado para cargas simples e volumes moderados de dados
- Não é ideal para atualizações em tempo real ou grandes volumes de dados

### Streaming de Dados

- Para atualizações em tempo real, considere usar tecnologias de streaming
- Apache Kafka é uma opção popular para streaming de dados

### Possibilidades Futuras

- Conexão em tempo real entre Excel e banco de dados
- Atualizações automáticas conforme os dados são modificados na fonte

```
NOTA: Técnicas avançadas como streaming de dados e atualizações em tempo real são tópicos para cursos mais avançados de ETL (Extração, Transformação e Carga).
```
