## **4. Passando um Prompt Avançado para o My GPT**

### **O que é um GPT**

Um GPT ou MyGPT é um Modelo de Chat GPT que pode ser customizado com instruções personalizadas e conhecimento específico.

### **Objetivo da Aula:**

Passar o Prompt Avançado para um GPT customizado para executar a Análise Sintópica de forma ainda mais poderosa.

### **Criar um GPT**

**Como criar um GPT customizado a partir de um Prompt Avançado:**

- **Adicionar os Componentes do Prompt Avançado para as Instruções do GPT**
- Adicionar Menu de Hotkeys – Atalhos – para facilitar a navegação.
- Adicionar informação complementar sobre o assunto (Leitura Sintópica).

### **Comparar o Prompt Avançado em 3 níveis diferentes:**

- ChatGPT 3.5
- ChatGPT 4.0
- MyGPT Customizado
