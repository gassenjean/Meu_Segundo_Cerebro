O Prompt Avançado é uma ferramenta importante para obter respostas precisas e úteis de uma IA (Inteligência Artificial). Aqui está uma visão geral do que é um Prompt Avançado e como ele pode ser estruturado:

**O que é um Prompt Avançado?**

Um Prompt Avançado é uma instrução detalhada que fornece informações adicionais para a IA gerar uma resposta precisa e relevante. Ele é projetado para fornecer mais contexto e clareza sobre o que você está pedindo, o que ajuda a IA a entender melhor o que você precisa.

**Estrutura de um Prompt Avançado**

Um Prompt Avançado pode ser estruturado da seguinte forma:

Dados + Contexto → Ação + Passos → Formato + Exemplo (da Saída)

* **Dados + Contexto**: Fornece informações adicionais sobre o assunto ou problema que você está tentando resolver.
* **Ação + Passos**: Descreve a ação que você deseja que a IA execute e os passos que ela deve seguir para gerar a resposta.
* **Formato + Exemplo (da Saída)**: Especifica o formato que você deseja que a resposta seja apresentada e fornece um exemplo de como a resposta deve ser.

**Objetivos de um Prompt Avançado**

Os objetivos principais de um Prompt Avançado são:

1. Forçar você a ter clareza sobre o que você está pedindo.
2. Melhorar o entendimento da IA e a qualidade da resposta.
3. Saber onde do Prompt colocar cada informação.
4. Facilitar a edição e customização do prompt.
